# 职场与修炼的荒诞平衡

在996高压职场环境下维持术师修炼的生存策略，也是本作核心主题。
核心矛盾：能力越强 → 效率越高 → 越容易暴露 → 越需要克制。
荒诞感：别人的觉醒是拯救世界，陈明的觉醒是更高效地996。

## 规则

- 通勤时间用于冥想修炼（地铁上闭目养神不会引起怀疑）
- 上班时间低调使用能力提升效率（不能太明显，需要"演"得自然）
- 下班后在出租屋独处时练习控制（唯一安全环境）
- 效率提升不能过于夸张，否则引来怀疑（赵磊已经开始注意到）
- 绩效提升是副产品，但也是掩护（林月加绩效 = 认可 ≠ 怀疑）

## 特征

- 核心矛盾：能力越强 → 效率越高 → 越容易暴露 → 越需要克制
- 荒诞感：别人的觉醒是拯救世界，陈明的觉醒是更高效地996
- 日常场景中的微妙张力：茶水间悬浮马克杯、会议室失控、卫生间偷练
- 程序员思维的独特优势：debug意识 → 能力控制 → 系统化修炼
