# 《一日之尊》世界观设定

---

## 一、世界背景

### 1.1 世界概述

本世界为**苍澜界**，一个以修仙问道为核心的仙侠世界。

天地初开，清气上升为天，浊气下沉为地，其间衍生万物。天地间弥漫着一种名为**灵气**的能量，修炼者吸纳灵气、淬炼肉身、凝练神魂，可突破凡人桎梏，蜕变为呼风唤雨、移山倒海的修仙者。

此界广袤无垠，划分为**九州四海**：
- **九州**：中州、东渊、南离、西荒、北冥、东南海域、西南云泽、东北幽岚、西北极寒
- **四海**：东海、南海、西海、北海

各州宗派林立，凡人国度与修仙势力交织，共存于这片天地之间。

### 1.2 故事舞台

主角所在的宗门为**青云宗**，位于**南离州**边缘地带的一座灵山——**青岚山**之上。

青岚山方圆百里灵气尚算充沛，但相较于中州大派、洞天福地，只能算是末流宗门。然而正因其不起眼，反而让主角有了从容布局、闷声发展的空间。

## 二、核心设定

### 2.1 时间轮回机制

**触发条件**：主角于某日酉时三刻（约下午五点四十五分）遭遇不明袭击后，陷入轮回。

**轮回规则**：

| 规则 | 说明 |
|------|------|
| **重置范围** | 时间倒转至当日卯时初刻（约早上五点），所有人物状态、建筑、物品恢复初始 |
| **记忆保留** | 主角保留完整记忆，包括所有经历、知识、对话细节 |
| **修为保留** | 主角在轮回期间的修炼突破、境界提升完全保留 |
| **物品限制** | 随身物品中，除**活物**（灵宠、灵植种子等）与**本命之物**外，其余物品全部重置 |
| **他人记忆** | 所有NPC不保留任何轮回记忆，对话、交易等均回到初始状态 |

**轮回触发**：每次死亡（包括意外、寿尽、自裁）或时间到达次日卯时初刻，轮回自动触发。

**关键细节**：
- 轮回并非瞬发，从死亡到意识回归约有三个呼吸的黑暗期
- 若在轮回过程中施法干扰（如护体灵符激活），可能造成记忆短暂模糊
- 主角若处于特殊阵法、秘境中，轮回时会被强制拉回青岚山寝舍

### 2.2 修仙境界体系

本世界修仙境界共分为**九个大境界**，每个大境界分为**初期、中期、后期、巅峰**四个小阶段：

| 境界 | 特征 | 寿元 | 社会地位 |
|------|------|------|----------|
| **炼气期** | 引气入体，凝聚法力，可御物飞行 | 120岁 | 外门弟子、普通散修 |
| **筑基期** | 丹田化海，法力凝实，可御剑杀敌 | 300岁 | 内门弟子、宗门长老 |
| **金丹期** | 丹成无悔，金丹不朽，可移山填海 | 800岁 | 宗门长老、世家家主 |
| **元婴期** | 元神出窍，神魂不灭，可撕裂虚空 | 2000岁 | 大派掌门、一方枭雄 |
| **化神期** | 炼神返虚，神识笼罩万里 | 5000岁 | 圣地之主、传说人物 |
| **炼虚期** | 虚实转换，化身天地 | 10000岁 | 顶级势力底蕴 |
| **合体期** | 万法归一，己身即道 | 20000岁 | 上古大能 |
| **大乘期** | 半步飞升，天人交感 | 50000岁 | 传说级存在 |
| **渡劫期** | 渡劫飞升，超脱此界 | 理论上无限 | 仙界存在 |

> **说明**：主角初始修为为**炼气三层**（最底层），在有限的轮回中逐步突破是核心看点之一。

### 2.3 修炼百艺

修仙者除自身境界提升外，还需掌握各类辅助手段：

| 技艺 | 简介 | 重要性 |
|------|------|--------|
| **炼丹** | 炼制丹药，辅助修炼、疗伤、突破 | 极高 |
| **炼器** | 炼制法宝、飞剑、阵盘 | 高 |
| **布阵** | 布置阵法，攻防辅助 | 高 |
| **符箓** | 绘制灵符，储存法术 | 中 |
| **灵植** | 培育灵药、灵草 | 中 |
| **御兽** | 驯养灵兽、灵宠 | 低 |

主角虽为稳健流，但至少需掌握**基础炼丹**与**基础布阵**，否则难以在轮回中建立有效优势。

### 2.4 世界规则

- **天地法则**：此界天道运转，修仙者需顺应天时、借用地利
- **因果业力**：杀孽过重者会沾染业力，影响修炼与渡劫
- **灵气潮汐**：每日不同时辰灵气浓度不同，卯时与酉时为灵气交汇点
- **雷劫**：金丹以上突破时需渡雷劫
- **禁地**：各处存在上古禁地、遗迹，其中蕴含机缘与危险

## 三、主要势力

### 3.1 青云宗（主角所在宗门）

**定位**：南离州三流宗门，传承约两千年，历史上有过辉煌，现已衰落

**实力**：

| 层次 | 人数 | 修为 | 代表人物 |
|------|------|------|----------|
| 掌门 | 1人 | 金丹后期 | 玉清真人 |
| 长老 | 7人 | 金丹/筑基巅峰 | 玄明长老、玄尘长老等 |
| 内门弟子 | 约50人 | 筑基期 | 若干 |
| 外门弟子 | 约300人 | 炼气期 | 若干 |

**主要建筑**：

- **青云殿**：宗门议事大殿
- **藏经阁**：存放功法典籍
- **炼丹房**：宗门炼丹之地
- **演武场**：弟子切磋、演法之处
- **灵药园**：培育灵药
- **弟子寝舍**：分内门、外门区域

**内部派系**：

- **玉清派**：掌门嫡系，势力最大
- **玄明派**：玄明长老为首，主张扩张
- **清修派**：不问俗事，专注修炼

### 3.2 外部势力

| **散修联盟** | 各州均有 | 分散 | 利益往来 | 主角可借力的势力 |

### 3.3 隐藏势力

**轮回之谜**：与主角轮回相关的不明势力，目前仅知曾在主角遇袭时出现过，身份不明。

## 四、主角设定

### 4.1 基础信息

| 项目 | 内容 |
|------|------|
| **姓名** | 陆沉（可调整） |
| **身份** | 青云宗外门弟子 |
| **年龄** | 18岁 |
| **初始修为** | 炼气三层 |
| **性格** | 沉稳、谨慎、深谋远虑，略带吐槽属性 |
| **核心特质** | 稳健至上，万事留后手，绝不浪 |

### 4.2 穿越者设定

主角为**地球穿越者**，灵魂夺舍了这具身体。

穿越带来的优势：

- **前世记忆**：保留地球世界的知识与思维方式
- **元神强度**：穿越过程中灵魂经历淬炼，神识略强于同阶修士
- **独特视角**：不受此界固有思维束缚，善于发现规则漏洞

### 4.3 金手指解析

**轮回机制**本身即为金手指，但需合理运用：

**优势利用**：

- 无限试错：任何危险决策都可以"死一次"重来
- 时间压缩：一次轮回可完成数日的准备工作
- 信息差：掌握所有NPC的行动模式与弱点
- 修为积累：每次轮回的修炼成果永久保留

**限制与挑战**：

- 仅有一日可用，时间极度有限
- 修为起点极低，难以正面对抗高阶敌人
- 轮回真相未明，不知何时能破解
- 修为越高，突破所需资源越多，一日之内难以满足

### 4.4 主角成长路线（初期规划）

| 阶段 | 目标修为 | 核心能力 | 关键行动 |
|------|----------|----------|----------|
| 第一阶段 | 炼气巅峰 | 基础炼丹、布阵 | 收集材料、拜师学艺 |
| 第二阶段 | 筑基期 | 炼制法器、基础符箓 | 建立后手、积累资源 |
| 第三阶段 | 金丹期 | 完整战力体系 | 探索轮回真相、破解循环 |

### 4.5 主角行事风格

- **绝不正面硬刚**：能用药解决的问题绝不动手，能用计解决的问题绝不冒险
- **后手为王**：每次行动至少准备三套预案，留有退路
- **信息优先**：行动前必先收集情报，不打无把握之仗
- **吐槽与自嘲**：以幽默化解压力，但不影响决策

## 六、核心冲突

### 6.1 表层冲突：轮回的枷锁

**表现**：主角困于一日轮回，看似拥有无限可能，实则处处受限。

**核心问题**：

- 如何在有限的一天内最大化收益？
- 如何应对宗门内的明枪暗箭？
- 如何积累足够的资源应对未来？

**看点**：极限操作、资源分配、时间管理

### 6.2 中层冲突：宗门的危机

**表现**：青云宗表面平静，实则暗流涌动。

**具体危机**：

| 危机 | 来源 | 威胁程度 |
|------|------|----------|
| 玄月谷打压 | 外部势力 | 中 |
| 玄明长老的野心 | 内部派系 | 高 |
| 灵脉衰退 | 自然因素 | 低（长期） |
| 内鬼泄密 | 不明 | 高（隐藏） |

**看点**：权谋斗争、派系倾轧、情报战

### 6.3 深层冲突：轮回的真相

**核心谜团**：

- 是谁在酉时三刻袭击了主角？
- 轮回的机制是什么？
- 能否破解？破解后会发生什么？

**与主角的关联**：轮回似乎是专门针对主角设下的陷阱，而非自然现象。

**看点**：悬疑解密、真相探索、隐藏Boss

### 6.4 终极冲突：天道与命运

**伏笔层面**（后期可能涉及）：

- 轮回是否与天道法则有关？
- 主角的穿越是否是某种安排？
- 修仙的终极目的是什么？

## 八、补充说明

### 8.1 风格融合

本书应融合以下元素：

- **稳健流**：《师兄稳健》的谋略风格，但可适度降低"苟"的程度
- **时间循环**：《土拨鼠之日》式的困境突破
- **吐槽修仙**：《大王饶命》的幽默感，但更内敛
- **日常修仙**：《凡人修仙传》的细致感

### 8.2 叙事建议

- 第一人称或第三人称均可，建议**第三人称有限视角**
- 每章开头可加入主角的内心独白或吐槽
- 轮回节点可作为章节分隔点
- 关键决策时刻可展示主角的"后手清单"
