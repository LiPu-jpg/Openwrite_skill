# 语言风格 (Language) —《谁让他修仙的》

## 用词风格：文白混杂

### 文言/史书风格（叙述世界观时）
- "问道宗，位列中央大陆五大仙门之一，修仙大能不计其数，强盛至极" [批次0]
- "孟家先祖率领军队历经千年，征伐了半个中央大陆" [批次2]
- "修仙一途道阻且长，练气、筑基、金丹、元婴、化神、炼虚、合体、渡劫" [批次0]

### 网络用语/口语化（角色对话/内心戏）
- "妈诶，修仙真刺激" [批次0]
- "妈的，神经病啊" [批次2]
- "去你大爷的龙凤和鸣！" [批次0]
- "呸，你当你是皇帝啊，还上意" [批次5]
- "狗屁的上意，给你这胖子脸了！" [批次5]

**特点**：两种风格快速切换，不追求统一，形成反差喜剧效果

## 句式偏好

### 短段为主（对话/动作/吐槽）
- 占比：47% [批次0] → 55% [批次2] → 60% [批次5]
- **平均占比：54%**
- 1-2句构成一段
- 例："碰碰运气而已。" / "谁不想进入问道宗。" / "不知道，孟兄你呢？" [批次0]
- 例："可以。" / "去。" / "没错。" [批次2]

### 长段用于世界观讲解
- 占比：3% [批次0] → 15% [批次2]
- **平均占比：9%**
- 5-8句构成一段
- 例：问道宗介绍（第1章）、灵根讲解（第2章）、中央大陆地理（第9章） [批次0]
- 例：孟家历史（第27章）、上古大世（第29章）、药园介绍（第33章） [批次2]

### 括号式补充 [批次2]
- 正文叙述后用括号补充角色真实想法，形成双层文本
- 例："（本来古月居士是不叫古月居士的，他姓胡...）"
- 例："（这已经不是他第一次听古月居士的故事了，每次都有出人意料的结局）"
- 例："（陆阳怀疑大师姐在耍自己，但他还是没有证据）"

## 修辞习惯

### 反预期比喻 [批次2]
- 用日常事物解构高大上概念
- "云芝像是一名撑天的巨人" → "陆阳像是鼹鼠从地里钻出来"
- "跟拔萝卜似的"（形容云芝把陆阳从土里拔出来）

### 高密度命名+即时解释 [批次0/2]
- 大量专有名词（灵根/功法/地名/草药），但通过角色对话自然解释，不堆砌

## 幽默体系

### 1. 身份反差喜剧 [批次0/2/5]
- 高大上身份+日常化行为
- 帝城孟家天才→孟景舟不正经 [批次2]
- 药王活了万年→人参娃娃舍不得送礼物 [批次2]
- 女魔头→大师姐云芝 [批次2]

### 2. 先甜后打脸反转 [批次0]
- 先建立读者预期再颠覆
- 孟景舟花重金买考题→云芝是考官，全程听到作弊计划
- 孟景舟："我这辆马车乃是异宝，就算是那些修仙大能也无法用神识探查到这里"→云芝全程在车里

### 3. 误会堆叠喜剧 [批次5]
- 通过信息不对称制造连环误会，双方各自脑补合理化
- 陆阳三人冒充考官→舵主误认为"减少竞争对手的魔道思维"
- 陆阳变大师姐→教主解读为"可遇不可求的天才"
- 秦元浩死因→传言"精力耗尽而亡"

### 4. 反预期对话落点 [批次0/2/5]
- 对话不按常理出牌，A问题B答非所问或故意曲解
- 陆阳："云芝姑娘你……" / 云芝：（轻迈脚步，白莲托起，飞向问道宗弟子中间） [批次0]
- 陆阳："你的脸怎么这么白？" / 云芝："孟景舟问我你吃的最多的是什么..." [批次2]
- 巴大爷："你谁？" / 陆阳："陆阳。" / 巴大爷："没问你叫什么，问的是你身份。" [批次2]

### 5. 歪楼式吐槽 [批次5]
- 吐槽落点故意与触发点不在同一维度
- 听到死刑→不吐槽残酷→吐槽怎么可以不加班
