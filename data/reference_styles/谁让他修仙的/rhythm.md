# 节奏风格 (Rhythm) —《谁让他修仙的》

## 段落长度分布

### 跨批次统计

| 段落类型 | 批次0 | 批次2 | 批次5 | 平均值 |
|----------|-------|-------|-------|--------|
| **短段** (1-2句) | 47% | 55% | 60% | **54%** |
| **中段** (3-4句) | 50% | 30% | - | **40%** |
| **长段** (5-8句) | 3% | 15% | - | **9%** |

### 趋势分析
- **短段占比逐渐增加**（47% → 55% → 60%），节奏越来越快
- 短段用于：对话、动作、吐槽、场景切换
- 长段用于：世界观讲解、历史背景、设定介绍

### 典型短段示例
- "碰碰运气而已。" [批次0]
- "谁不想进入问道宗。" [批次0]
- "可以。" / "去。" / "没错。" [批次2]

### 典型长段示例
- 问道宗介绍（第1章）[批次0]
- 灵根讲解（第2章）[批次0]
- 孟家历史（第27章）[批次2]
- 上古大世（第29章）[批次2]
- 药园介绍（第33章）[批次2]

## 场景切换方式

### 强制断点 [批次0/2/5]
- **每章结尾用"（本章完）"打断**，不追求自然过渡
- 章节平均字数：2026 [批次0] → 2200 [批次2] → **平均2113字**
- 场景转换频率：12次/13章 [批次0] → 17次/14章 [批次5] → **平均每章1.2次**

### 直接跳转 [批次2]
- 无过渡，直接进入新场景
- 例："问道宗内有很多产业"（第26章）
- 例："三天后"（第30章）

### 知识嵌入式推进 [批次2]
- 每个场景都承载世界观讲解，通过角色互动而非旁白灌输
- 百香楼点菜→菜单介绍食材
- 药园松土→人参娃娃讲草药
- 孟景舟身份揭晓→引出帝城孟家历史

## 紧张-松弛节奏模式

### 高密度信息+幽默消解 [批次2]
1. **紧张**：先用长段讲解复杂设定（孟家历史/上古大世）
2. **松弛**：紧接着用短段吐槽打破严肃感（"妈的，神经病啊"）
3. 形成"紧张-松弛"循环，避免信息过载

### 考验-通关-评价循环 [批次0/5]
1. **紧张**：考验规则说明（长段）
2. **松弛**：标准答案示范（蛮骨，短段）
3. **紧张**：非常规解法（陆阳/孟景舟，中段）
4. **松弛**：权威震惊反应（短段吐槽）

## 节奏控制技巧

### 1. 短段密集轰炸
- 连续3-5个短段（1句话），制造快节奏
- 用于：紧张时刻、对话交锋、连续动作

### 2. 长段缓冲
- 插入1个长段（5-8句），放慢节奏
- 用于：世界观讲解、历史回顾、设定介绍

### 3. 吐槽打断
- 严肃叙述后立刻插入短段吐槽
- 防止长段过于沉闷

### 4. 强制断点
- 不管情节是否完结，章节结束就打断
- 制造悬念，保持阅读节奏
