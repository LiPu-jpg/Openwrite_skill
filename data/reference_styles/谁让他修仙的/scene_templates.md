# 场景模板

> 基于3组批次提取结果合并（批次0: 第1-13章 / 批次2: 第26-39章 / 批次5: 第63-76章）

## 反复出现的场景结构模式

### 1. 考验-通关-评价三段式 ★★★（出现频率：2/3批次）

**固定结构**：
```
1. 考验规则说明（权威角色讲解）
2. 标准答案示范（蛮骨诚实通关）
3. 非常规解法（陆阳/孟景舟钻空子）
4. 权威震惊反应（暴怒/强行解释）
```

**证据**：

#### 批次0：问道宗三关考验
- **第一关测灵根**：戴不凡把脉 → 陆阳剑灵根、孟景舟纯阳灵根 → 众人热切
- **第二关诚实**：河灵三把斧头 → 蛮骨诚实回答 → 陆阳把斧头都扔河里 → 孟景舟跳河变三个自己抢斧头 → 河灵暴怒
- **第三关问心山**：戴不凡设计 → 众人苦爬 → 陆阳孟景舟造梯子 → 戴不凡暴怒

#### 批次5：不朽教三关考验
- **第一关意志力**：蛮骨从蜘蛛学坚持+螳螂学计谋（标准答案）→ 孟景舟被蜘蛛咬获得能力，直接斩首敌将 → 陆阳观察螳螂悟出螳螂拳，变身螳螂平推 → 副教主："谁家螳螂拳是这么打的？！"
- **第二关做事周到**：土豆任务 → 孟景舟把卖土豆大爷叫到门口 → 陆阳买下土豆车写"土豆不卖" → 舵主认可为"魔道思维"
- **第三关本性释放**：仙眼考验 → 陆阳冒充考官提前淘汰竞争者 → 舵主："这仨一看就知道不是好东西！"（误判为优秀）

**模式特点**：
- 蛮骨永远是标准答案示范
- 陆阳/孟景舟永远钻空子
- 权威永远震惊/暴怒/强行解释

### 2. 先甜后打脸反转 ★★（出现频率：1/3批次）

**固定结构**：
```
1. 建立预期（角色自信满满）
2. 信息揭示（读者/叙述者知道真相）
3. 角色行为（基于错误信息）
4. 反转揭晓（真相大白）
5. 吐槽/解构（叙述者评论）
```

**证据**：

#### 批次0：云芝是考官
- 孟景舟花重金买考题 → 云芝是考官，全程听到作弊计划
- 陆阳和孟景舟商量作弊对策 → 云芝："让大家久等了，我是问道宗大师姐，本次考验主持者"
- 孟景舟："我这辆马车乃是异宝，就算是那些修仙大能也无法用神识探查到这里" → 云芝全程在车里
- 叙述者："完了，他们居然在考官面前商量如何作弊"

**模式特点**：
- 先知后行（读者先知道真相）
- 戏剧反讽（角色不知情）
- 叙述者与读者共谋

---

### 3. 身份反差喜剧 ★★（出现频率：1/3批次）

**固定结构**：
```
1. 高大上预期建立（史书笔调/权威介绍）
2. 日常细节解构（角色行为/对话）
3. 反差落点（吐槽/评论）
```

**证据**：

#### 批次2：帝城孟家
- 预期：孟家先祖率领军队历经千年，征伐了半个中央大陆
- 解构：孟景舟："想不到你们猜出来了我就是孟家那个横空出世的天才"
- 落点：陆阳："不，我们没觉得你是天才"

#### 批次2：药王群体
- 预期：药王活了不知多少岁月
- 解构：人参娃娃："我这老参须你带走，哦，这个太值钱了我舍不得"
- 落点：性格像小孩

#### 批次2：女魔头
- 预期：女魔头令人发指
- 解构：云芝：大师姐
- 落点：陆阳利用这个身份拉近关系

**模式特点**：
- 史诗→草率
- 传说→日常
- 权威→解构

---

### 4. 知识嵌入式场景 ★★（出现频率：1/3批次）

**固定结构**：
```
1. 场景设置（日常活动）
2. 角色互动（对话/行动）
3. 知识讲解（自然嵌入）
4. 剧情推进（讲解即行动）
```

**证据**：

#### 批次2：百香楼点菜
- 场景：陆阳等人进入百香楼
- 互动：查看菜单
- 讲解：菜单介绍各种食材背景
- 推进：点菜过程展现世界观

#### 批次2：药园松土
- 场景：陆阳在药园干活
- 互动：人参娃娃边走边讲
- 讲解：草药知识
- 推进：建立陆阳与药王关系

#### 批次2：孟景舟身份揭晓
- 场景：孟景舟暴露身份
- 互动：众人反应
- 讲解：引出帝城孟家十万年历史
- 推进：孟景舟角色深化

**模式特点**：
- 不用旁白灌输
- 通过角色互动解释
- 讲解过程本身也推进剧情

---

### 5. 权威叙述+即时解构 ★（出现频率：1/3批次）

**固定结构**：
```
1. 权威叙述（文言/史书风格）
2. 即时解构（细节/吐槽）
3. 反差落点（喜剧效果）
```

**证据**：

#### 批次2：夏帝先祖和孟家先祖决战
- 权威：夏帝先祖和孟家先祖决战星空，日月无光
- 解构：孟家先祖愿赌服输，拱手让位（过于简单）
- 落点：史诗→草率

#### 批次2：岁月终焉花
- 权威：岁月终焉花只在世界末日绽放
- 解构：我记得我是很和平的时候开花的，大概是谣传吧
- 落点：传说→日常

#### 批次2：八宝莲花
- 权威：八宝莲花名称由来的各种学术猜想
- 解构：古月居士看了一眼手里的八宝粥，随口就说不如叫八宝莲花
- 落点：学术→随意

**模式特点**：
- 先建立权威感
- 紧接着用细节打破
- 制造喜剧反差

---

### 6. 多重通关路径设计 ★（出现频率：1/3批次）

**固定结构**：
```
1. 考验设置（有标准答案）
2. 标准路径（蛮骨示范）
3. 非常规路径A（孟景舟）
4. 非常规路径B（陆阳）
5. 权威评价（震惊/强行解释）
```

**证据**：

#### 批次5：蜘蛛螳螂考验
- 考验：意志力测试
- 标准：蛮骨从蜘蛛学坚持+螳螂学计谋
- 路径A：孟景舟被蜘蛛咬获得能力，直接斩首敌将
- 路径B：陆阳观察螳螂悟出螳螂拳，变身螳螂平推
- 评价：副教主："谁家螳螂拳是这么打的？！"

**模式特点**：
- 同一考验多种解法
- 展现角色不同思路
- 权威震惊反应

## 开场/转折/高潮/收尾套路

### 开场套路

#### 1. 世界观介绍开场 [批次0]
**结构**：权威叙述 → 角色登场 → 吐槽

**证据**：
- 问道宗介绍（第1章）→ 陆阳孟景舟相遇 → "妈诶，这可比我们孟家的宅子霸气多了"

#### 2. 考验规则说明开场 [批次0/5]
**结构**：权威角色讲解 → 众人反应 → 考验开始

**证据**：
- 戴不凡："此物名为问心山" → 众人苦爬 → 陆阳孟景舟造梯子

### 转折套路

#### 1. 信息揭示转折 [批次0]
**结构**：角色自信 → 真相揭示 → 反转

**证据**：
- 孟景舟："马车是异宝" → 云芝全程在车里 → "完了"

#### 2. 权威视角转折 [批次5]
**结构**：主角行为 → 切换到权威视角 → 误判/强行解释

**证据**：
- 陆阳变身螳螂 → 副教主："谁家螳螂拳是这么打的？！" → 教主："武道天赋极高"

### 高潮套路

#### 1. 非常规通关高潮 [批次0/5]
**结构**：标准答案示范 → 非常规解法 → 权威震惊

**证据**：
- 蛮骨诚实回答 → 陆阳把斧头都扔河里 → 河灵暴怒

#### 2. 误会堆叠高潮 [批次5]
**结构**：角色行为 → 权威误判 → 双方脑补合理化

**证据**：
- 陆阳冒充考官 → 舵主："魔道思维" → 认可为优秀

### 收尾套路

#### 1. "本章完"强制断点 [批次0/2/5]
**结构**：场景未完 → （本章完） → 制造悬念

**证据**：
- 第1-76章每章结尾

#### 2. 吐槽式收尾 [批次0/5]
**结构**：事件结束 → 叙述者评论 → 轻松收尾

**证据**：
- "说好的老实人呢？"
- "想必秦元浩九泉之下知晓此事，也能瞑目了"

## 场景模板的稳定性

### 跨批次一致元素
- 考验-通关-评价三段式（2/3批次）
- "本章完"强制断点（3/3批次）
- 权威震惊反应（3/3批次）

### 演化趋势
- **批次0→2**：从单纯反转到身份反差喜剧
- **批次2→5**：从知识嵌入到多重通关路径
- **权威视角增加**：批次5出现8次权威视角转换

### 核心不变量
无论章节进展，场景模板始终保持：
1. 考验-通关-评价结构
2. 标准答案+非常规解法
3. 权威震惊/暴怒/强行解释
4. "本章完"强制断点
