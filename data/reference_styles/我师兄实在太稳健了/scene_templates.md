# 场景模板 (Scene Templates)

> 记录《我师兄实在太稳健了》中反复出现的场景结构模式。

## 模板1：以退为进说服场景

### 结构
```
对方带着问题/敌意登场 → 李长寿示弱/示诚 → 陈述利害（非威胁） → 对方动摇 → 借势提出请求 → 对方主动答应
```

### 特征
- 李长寿**从不摆出强者姿态**，始终以"晚辈""不敢"等措辞
- 核心技术是**利害陈述**——不是说服对方相信自己，而是让对方自己算清利弊
- 适度暴露"私心"增加可信度——「说实话，晚辈也并非毫无私心」
- 说服结束后往往**追加一个小请求**——因为对方已在心理上认可了他

### 示例结构（说服黄龙真人·第233章前后）
1. 黄龙真人带着质疑登场：被燃灯挑拨，质问南海海神
2. 李长寿不辩解，而是以三连反问分析西方教的真实目的
3. 陈述龙族若归西方的利害：族运全面崩溃
4. 黄龙真人面色变化→深深道揖致歉
5. 李长寿主动承认自己有私心→增加信任
6. 追加请求：让黄龙真人帮忙散播打油诗
7. 黄龙真人一喜→立即执行

## 模板2：小琼峰日常场景

### 结构
```
李长寿回到小琼峰/日常开始 → 灵娥迎接/互动 → 讲道/做饭/对话 → 温暖细节 → 偶有来客打扰 → 温馨收尾
```

### 特征
- 小琼峰是全书的**情感锚点**——无论外面多危险，这里永远温暖
- 灵娥的存在是温馨感的核心——崇拜师兄、活泼好奇
- 李长寿的**讲道**段落既传递设定也展现师兄人格魅力
- 来客（酒乌、酒玖、江林儿、有琴玄雅等）打扰带来新信息或支线
- 以人间烟火（做饭/烤鱼/打牌）收场

### 示例结构（讲道+烤灵兽·第233章后半）
1. 李长寿在湖边草屋为灵娥和熊伶俐讲道
2. 深入浅出的比喻+灵娥崇拜眼神
3. 熊伶俐问"为什么小琼峰灵兽比外面好吃"
4. 李长寿借机讲"道藏于碗筷之中"
5. 灵娥痴迷看师兄→"她就喜欢师兄随便一件事都能扯这么多道理的亚子"
6. 夜幕降临→现场烤灵兽→酒乌来访→温馨结束

## 模板3：天庭政治/审判场景

### 结构
```
正式场合开启 → 各方就位 → 前期铺垫/证人问询 → 核心辩论 → 反转/证据展示 → 裁决/收场
```

### 特征
- 高度**程序化**的叙述——仿古代庙堂/近代法庭
- 多方势力在场：玉帝、李长寿、被告（燃灯）、旁观仙神
- 李长寿作为**控场者**引导审理节奏
- 穿插旁观者反应营造氛围（如太乙真人被封嘴仍嗡嗡评论）
- 玉帝与李长寿的**暗中默契**——表面各司其职，实际早已商定

### 示例结构（凌霄殿审燃灯·第615章）
1. 凌霄殿正式开庭→玉帝端坐→仙神两侧
2. 道微子出庭作证→先承认自身过错
3. 李长寿追问核心：是否有人指使？
4. 玉帝降天罚轰雷恫吓道微子说真话
5. 道微子吐出燃灯副教主的名字
6. 燃灯辩护：转移矛盾→李长寿反驳
7. 小文吏王善仗义执言→玉帝当场提拔
8. 物证（留影球）展示→全场情绪推高
9. 燃灯认罪→量刑

## 模板4：棋局揭示场景

### 结构
```
读者/角色以为事件已结束 → 新信息揭示 → 发现此前的某个小动作原来有深意 → 棋局全貌浮现 → 李长寿的稳健得到确认
```

### 特征
- 棋局揭示是本书的**核心爽点机制**
- 伏笔可能在几十甚至几百章前埋下
- 揭示方式：通过**第三者视角**回看李长寿之前的行为
- 元始天尊的评价常作为棋局复盘的权威注释：
  > "长庚跟脚清正、福源深厚，既懂变通、又善周全……"

### 示例（散播打油诗→西方教忌惮增加）
1. 前期：李长寿让黄龙真人帮散播南海海神的打油诗
2. 中期：读者以为这只是小事
3. 后期：西方教因为种种迹象（人教、阐教、截教都与海神有联系）而忌惮增加
4. 揭示：李长寿的策略分析——三教保护伞的烟雾弹

## 模板5：纸道人行动场景

### 结构
```
任务目标确定 → 纸道人出击（各种型号）→ 暗中行动 → 完成任务 → 本体检讨/调整
```

### 特征
- 纸道人是李长寿行事的**代理系统**——本体极少亲自出面
- 不同型号纸道人有不同的功能和"名号"（人教小法师型等）
- 行动过程往往**被省略**，只展示结果
- 行动后李长寿的本体会进行"安全审计"——检查是否留下破绽

### 示例（终局·纸道人方阵vs天劫）
1. 天劫劈碎叁号假身
2. 袖中飞出备用假身→继续站立
3. 再被劈碎→从山林走出
4. 地脉/水源/树干中数万纸道人涌出
5. 组成百丈巨像→自爆冲入劫云
6. 劫云被炸散→李长寿争取到时间

## 模板6：师门故人重逢场景

### 结构
```
故人归来 → 初见/试探 → 私下交谈（李长寿掌握信息差）→ 李长寿引导故人情绪 → 给出好处/丹药 → 关系确立 → 回归日常
```

### 特征
- 李长寿对师门长辈/故人始终保持"比他们更了解局势"的信息优势
- 通过**精确的信息释放**（只说该说的，只在最佳时机说）掌握对话节奏
- 好处/丹药的给予是"收买人心"和"真情实意"的混合——读者分不清比例但不在意
- 重逢后的日常互动恢复温馨基调

### 示例（江林儿师祖回归·第144章）
1. 江林儿天仙突破后回度仙门
2. 李长寿与江林儿私下对话：揭露师伯转世一事、蒯思道人被杀真相
3. 江林儿情绪波动→自扇耳光→李长寿劝住
4. 李长寿给出疗伤丹药（价值远超江林儿的礼物）
5. 江林儿："你这样的朋友，我交定了！" / "师祖，我是您徒孙。"
6. 回归酒席日常→江林儿检查酒玖→全场笑闹

## 模板7：终极对弈/决战场景（末篇特有）

### 结构
```
底牌亮出 → 道祖压制 → 分身消耗 → 各方反应全景 → 新底牌 → 声势宣言 → 道祖反击 → 真正的最终手段
```

### 特征
- 与常规战斗不同，这是**道与道的碰撞**
- 渐进递升：从假身→纸道人方阵→百丈巨像→太极图传送
- 穿插各路大能的反应：赵公明、女娲、玉帝、元始天尊——营造"天地为之震动"
- 李长寿的声音传遍天地是关键的仪式性时刻
- 杨戬的锦囊线在此刻交汇
