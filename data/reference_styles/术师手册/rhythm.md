# 节奏风格 (Rhythm Style)

> 记录《术师手册》的段落结构、章节节奏和信息密度分布。

## 段落长度分布

| 类型 | 占比 | 使用场景 |
|------|------|----------|
| **短段**（1-2句） | ~60% | 吐槽、内心独白、对话反应、动作描写、紧张节奏 |
| **中段**（3-5句） | ~30% | 场景过渡、设定补充、旁观者评述 |
| **长段**（6句以上） | ~10% | 反派哲学独白、世界观核心设定、虚境宏观描写 |

### 短段的使用特征
- 吐槽段几乎全部为短段
- 战斗场景大量使用1句1段，制造速度感
- 内心独白经常独立成段，即使只有一个感叹词
  > 「是敌人，也是同路人，西莉亚第一时间就感觉出来了，哪怕对方一句话都还没说。」

### 长段的使用时机
- 伊古拉的"支配"哲学独白（第19章）
- 虚境世界观说明（第24章）
- 血月审判规则解释（第49-55章）
- 旁观者内心独白（西莉亚的利益计算）

## 章节结构

- **章节字数**：约800-1500字/章，极少超过2000字
- **章节标题**：以对话或角色行为命名，常带悬念或反转暗示
  > 「剑姬，还不可以休息哦」
  > 「你要在万众瞩目下死去」
  > 「可恶，让他装到了」
  > 「这么好看的脸，挨一拳能哭很久吧！」
- **章末钩子**：几乎每章结尾设置，形式包括：
  - 悬念问题（「该结束了。」）
  - 突然反转（新角色/新信息出现）
  - 危机前兆
  - 情绪高潮点

## 紧张-松弛节奏模式

### 基本循环
```
铺垫（设定/日常）→ 期望建立 → 打脸/反转 → 吐槽消化 → 新铺垫
```

### 战斗节奏
```
对峙 → 试探攻击 → 劣势/危机 → 意外转机 → 爆发逆转 → 战后吐槽
```
- 战斗中穿插旁观者评论、角色内心吐槽，打破纯粹的紧张感
- "十秒休息"等小间隔做节奏锚点

### 设定说明节奏（乒乓球模式）
```
信息点A → 角色反应 → 信息点B → 角色吐槽 → 信息点C → 角色震惊
```
- 每1-2个信息点后必须有角色反应
- 绝不连续灌输超过3段设定无反应
- 反应形式：吐槽、震惊、类比、追问

## 场景切换方式

### 1. 明确章节切换
- 使用 `← →` 章节标记
- 新章节标题

### 2. 章内场景分隔
- `......` + `...`（六点省略+三点省略）
- 空行分隔
- 时间/地点跳转标记

### 3. 视角切换锚定
- 切换时通常以新视角角色的**环境感知**开场：
  > 「繁星国度，迦乐世，剑与玫瑰术师大学。」（地点锚定）
  > 「血月国度，碎湖监狱，餐厅。」（地点锚定）

## 信息密度分布

| 场景类型 | 信息密度 | 特征 |
|----------|----------|------|
| 日常搞笑 | 低 | 以角色互动和吐槽为主 |
| 设定说明 | 高 | 乒乓球式紧凑输出 |
| 战斗场景 | 中-高 | 动作+心理+旁观者评论交叉 |
| 虚境探索 | 中 | 世界描写+角色对话+战利品系统 |
| 煽情段落 | 低 | 细节聚焦，留白多 |
| 血月审判 | 极高 | 规则说明+多角色反应+投票机制 |

## 节奏加速技巧

1. **连续短句/短段**制造速度感
2. **省略对话标签**（省去"某某说道"）
3. **动作直接呈现**（不加心理铺垫）
4. **交叉视角快切**（战斗中多角度快速切换）

## 节奏减速技巧

1. **环境描写插入**（虚境的宏观景象）
2. **哲学式独白**（伊古拉"支配"论述）
3. **回忆/嵌入式故事**（剑姬的过去）
4. **食物描写**（餐厅场景，珍馐列举）
