# 场景模板 (Scene Templates)

> 记录《术师手册》中反复出现的场景结构模式。

## 模板1：战斗场景

### 结构
```
对峙铺垫 → 试探交锋 → 劣势/危机 → 转机/成长 → 爆发逆转 → 战后吐槽/清算
```

### 特征
- 战斗中穿插**旁观者评论**和**角色内心吐槽**，打破纯紧张感
- **实时成长模式**：角色在战斗中飞速成长，多视角展现（自身体感+敌人困惑+旁观者震惊）
- 战斗后必有**战利品检索**环节（类游戏化）
- 受伤描写**轻量化**，侧重心理和战术而非血腥细节

### 示例结构（索妮娅 vs 绿皮兽人，虚境）
1. 遭遇兽人→发现苦弱派系克制→索妮娅刮痧无效
2. 兽人战斗经验丰富，缠斗消耗
3. 亚修主动当嘲讽随从牺牲自己
4. 索妮娅抓准机会释放"里波动剑"击杀
5. 战后检索「橡树皮肤」术灵→轻松对话→继续探索

## 模板2：日常搞笑场景

### 结构
```
平静日常铺垫 → 期望建立 → 打脸/反转 → 吐槽链 → 新的日常
```

### 特征
- 吐槽密度最高的场景类型
- 角色间的误会是核心笑点来源
- 多使用短段快节奏

### 示例结构（亚修在监狱找人打架）
1. 亚修决定找人打架获取经验
2. 找到光头壮汉→故意打翻牛奶
3. 期望壮汉发怒→壮汉却温柔擦鞋推荐牛奶
4. 亚修愣住→吐槽

## 模板3：煽情场景

### 结构
```
情感铺垫（细节堆叠）→ 情感爆发点 → 冷静转折/吐槽打断 → 余韵留白
```

### 特征
- 煽情**绝不超过3段**就会被吐槽或转折打断
- 用**细节**而非直接抒情传递情感
- 煽情后的吐槽不是消解情感，而是让情感更真实

### 示例（剑姬与亚修关于"主人"的对话）
1. 剑姬戳额头→逼问"你想说你是我的什么" →情绪爆发
2. 亚修冷静回应"我不是你的主人"→发誓
3. 剑姬安心→"走吧，去餐厅啊"→突然日常化

## 模板4：设定说明场景

### 结构（乒乓球模式）
```
信息点A → 角色反应 → 信息点B → 角色吐槽 → 信息点C → 角色震惊/类比
```

### 特征
- **绝不连续灌输超过3段设定无反应**
- 接收者的反应形式多样：吐槽、类比前世、震惊追问、歪楼跑题
- 信息提供者通常有鲜明个性（朗拿的温柔、索妮娅的冷静、医疗师的热情）

### 示例结构（朗拿解释死斗社规则）
1. 朗拿："死刑犯会被监狱榨干所有价值"
2. 亚修内心：这跟公司一样啊
3. 朗拿继续解释贡献度制度
4. 亚修吐槽：比他公司还差一点
5. 朗拿揭示血月审判
6. 亚修震惊

## 模板5：旁观者视角链

### 结构
```
低级评价（新手视角）→ 中级震撼（内行视角）→ 高级危机（利益相关者视角）→ 群体反应
```

### 特征
- 内心独白密度梯度：心机越深→独白越多
- 每个旁观者的评价维度不同
- 最终汇聚到群体反应，制造"全场震惊"效果

### 示例（索妮娅训练场景）
1. 英古莉特（剑术评价）：专业角度评估→震惊
2. 菲利克斯（天才认知）：不可思议的成长速度
3. 西莉亚（情敌视角）：大段利益计算+危机感
4. 全场学徒：围观+议论

## 模板6：死斗/擂台场景

### 结构
```
死斗规则说明 → 对手能力揭示（反派独白）→ 主角劣势 → 关键转机 → 逆转
```

### 特征
- 通过反派内心独白揭示能力原理（伊古拉的心灵术）
- 主角的转机来自出其不意而非硬实力碾压
- 旁观者的解说增加信息密度

## 模板7：虚境探索场景

### 结构
```
虚境进入 → 环境描写（知识之海）→ 探索/航行 → 遭遇（传承岛/知识生物）→ 战斗/考验 → 收获/撤退 → 返回讨论
```

### 特征
- 虚境描写比现实场景更**诗意**，用铅灰色大海、白雾等意象
- 探索过程带有**游戏化元素**（虚境探索、格子地图、白费力气/最好别去标记）
- 战利品系统化呈现（术灵名+限制+效果+备注）

## 模板8：社交陷阱/话术场景

### 结构
```
日常对话伪装 → 暗中布局（一方有隐藏目的）→ 试探/交锋 → 露出马脚/被识破 → 破局
```

### 特征
- 对话表面和谐但信息战激烈
- 读者通过内心独白知道双方真实意图
- 亚修使用前世社畜经验应对

## 模板9：血月审判场景

### 结构
```
候审铺垫（最后的晚餐）→ 审判规则说明 → 涤罪之焰考验 → 投票互动环节 →
犯人介绍/履历展示 → 票数攀升/行刑者成长 → 高潮（泰坦行刑者降临）→ 处刑
```

### 特征
- 全民直播形式，纳古作为主持人
- 融合了选秀节目+死刑执行+宗教仪式的多重元素
- 信息密度极高（规则+犯人背景+社会制度）
- 将社会制度反讽推到极致（"救赎积分兑换优惠券"）

## 模板10：角色冲突→和解场景

### 结构
```
期待→失望→冲突→底线碰撞→冷静反击→和解
```

### 特征
- 每个情绪节点都必须展开，不能跳过
- 和解不是大和解，而是达成某种新的平衡
- 冲突后的关系往往比之前更深入

### 示例（亚修与剑姬）
1. 亚修期望剑姬帮忙逃狱→被拒绝（失望）
2. 亚修说"你只是虚拟人物"（冲突）
3. 剑姬炸毛→底线碰撞（不是主人！）
4. 亚修冷静反击→平等对待承诺→发誓
5. 剑姬放心→"走吧去吃早饭"（和解→日常化）

## 模板11：制度/世界观讨论场景

### 结构
```
角色提出疑问 → 信息提供者解释制度 → 穿越者视角碰撞 → 价值观冲突 → 无解/留白
```

### 特征
- 亚修的现代价值观与血月国度制度碰撞
- 不给出明确的对错判断，而是让读者自行思考
- 信息提供者（如医疗师）代表本世界的"常识"视角

### 示例（亚修与医疗师讨论堕胎权/芯片监管）
1. 亚修提出现代价值观的疑问
2. 医疗师以本世界"常识"回应
3. 亚修发现自己变成了"自由恐怖主义者"
4. 无法反驳但不认同→留白
