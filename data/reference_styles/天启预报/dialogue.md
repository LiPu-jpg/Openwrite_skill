# 对话风格 (Dialogue Style)

> 记录《天启预报》的对话格式、角色语言个性、对话驱动模式和互动风格。

## 对话格式规范

### 基本格式
- **普通对话**：使用 `""` 双引号
  > "可以啊，小子。"那个粗豪的男人用力地拍了一下他的肩膀
- **内心独白**：直接融入叙述，无引号，或以反问句形式
  > 我是谁？我在哪儿？究竟发生了什么？
- **意识通话**：使用叙述描写标明（"通过意识通话询问"）
  > 槐诗通过意识通话询问藏在铃铛里的傅依："总感觉去了是做工具人……"

### 对话标签使用
- **动作标签为主**：用动作描写替代"说道"系列
  > 老肖尴尬地咳嗽了两声，回头看向自己的监察官：「你还看啥热闹——」
  > 他抬起眼眸，凝视着大屏幕上的名单，平静地说道
- **省略标签**频率高：快速对话中直接换行，不加任何标签
  > "什么意外？"
  > "……" → 直接以沉默表示
- **场景描写嵌入标签**：在对话间穿插环境/动作描写调节节奏
  > 艾晴沉默了，许久，没有说话，只是直勾勾地看着他。不知为何，忽然轻声笑起来。

### 格式分配规则
| 对话类型 | 格式 |
|----------|------|
| 角色说话（现实） | "" |
| 内心独白（POV角色） | 无引号融入叙述 |
| 意识通话/远程沟通 | "" + 叙述说明 |
| TRPG叙述（kp角色） | "" |
| 僧侣祈祷/仪式言语 | "" |

## 对话驱动模式

### 1. 冰山式对话（核心模式）
对话表面寡淡，信息量藏在水面之下：
```
艾晴："你的保证能够再早一点，我就不用被人说是女疯子了。"
（水面下：多年的创伤、愤怒、不愿示弱、对家族的复杂情感）
```
```
阴崖："想办法调到国外去吧，小晴。去伦敦，去罗马。"
（水面下：有无法明说的危险正在逼近、他的权力有限、这是最后保护她的机会）
```

### 2. 训话式长独白
长辈/权威角色的训话独白——不是对话交锋，而是单方面输出价值观和规则：
```
林家老人连续数段训话 → 中间穿插"——都听明白了吗？" → 齐声"明白！" → 继续
```
特征：排比递进、反问加强、霸道但有温情底色

### 3. deadpan式冷面对话
用极度平静/随意的语气说残酷或戏剧性的内容：
```
槐诗看着最后一个升华者："就这？没别的了？"
→ 在刚杀了一群人之后以困惑语气追问，制造不寒而栗的效果
```
```
"这个，送给你。" → 竖中指 → 直播结束
→ 以轻描淡写的语气做出极具冲击力的行为
```

### 4. TRPG式互动对话
模拟跑团的GM-玩家互动节奏：
```
kp："妙哉！简直是传奇一般的表现！"
乌鸦："那么，我们来商量一下接下来的事情吧……比方说，补偿？"
→ 评价→交涉→条件谈判的TRPG式推进
```

### 5. 温情伪装式对话
角色以调侃/淡然的方式传递关心，不直接表露情感：
```
大表哥："把琥珀那个小丫头看好，可别让她血气上头把鹿鸣馆的人给剁了。"
→ 表面是吐槽/命令，实际是保护和关心
```

### 6. 恳请-拒绝式对话
一方诚恳恳请，一方以冷静/讽刺拒绝，形成情感张力：
```
阴崖："家里很多事情我可以说了算，如果你能回到家里来——"
艾晴："干嘛和一个女疯子讲道理呢，大哥。"
→ 恳请越真诚，拒绝越锋利，情感张力越大
```

### 7. 虔诚-荒诞式对话
信徒/NPC的虔诚态度与主角的荒诞解读形成反差：
```
僧侣狂喜磕头
槐诗："……那你们国家岂不是很可怜？"
→ 僧侣越真诚，槐诗越困惑，荒诞感越强
```

## 角色对话个性

### 槐诗
- **口吻**：慵懒、冷面、偶尔钝感、偶尔残酷
- **沉默比**：高——大量场景中槐诗以沉默/行动代替对话
- **典型表达**：
  > "就这？没别的了？"（disappointment式杀人后感想）
  > "算我高看你们了……"（冷感叹息）
  > 静静竖中指，不说话
- **对话特征**：当他不说话时比说话时更有威慑力

### 乌鸦
- **口吻**：戏精式兴奋、优雅自得、促狭刻薄
- **称呼系统**：「我的书记官」（称槐诗）、「主持者们」（称kp等人）
- **典型表达**：
  > "哎呀哎呀，真是的，太粗鲁了点吧？"
  > "没想到他进步的速度这么快……"
  > 搓小翅膀、颔首满足
- **对话特征**：像戏剧演员一样享受每一个场景，语言有仪式感

### 艾晴
- **口吻**：冰冷精准、内藏裂痕、偶尔锋利如刀
- **典型表达**：
  > "你的保证能够再早一点，我就不用被人说是坐着轮椅的女疯子了。"
  > "和一个女疯子讲道理呢。"
  > "我现在生活的很好，有自己的工作和自己想要做的事情。"
- **对话特征**：用"女疯子"自称不是自嘲，而是反击——把对方的标签变成武器

### 大表哥
- **口吻**：粗豪直爽、江湖做派、吐槽型领导
- **典型表达**：
  > "见什么见，不见！让他们滚去羊城赛区——"
  > "把琥珀看好，可别让她把人剁了。"
  > 幸灾乐祸+抱怨+护短
- **对话特征**：管理风格像江湖大哥而非官僚

### 林家老人（烛龙）
- **口吻**：威严霸道、铁血温情、演讲式训话
- **典型表达**：
  > "就是白死了！连棺材钱都不出！"
  > "你们都有远大的前程。"
  > "林家就是你们的保命符！"
- **对话特征**：训话中残酷与温情交替，制造"铁血家长"的矛盾魅力

### kp
- **口吻**：TRPG主持人式兴奋、夸张赞叹
- **典型表达**：
  > "妙哉！简直是传奇一般的表现！传奇！"
  > 玩骰子、微笑、闪闪发光的眼神
- **对话特征**：用TRPG术语评价现实事件

### 诸红尘
- **口吻**：冷静分析、一针见血、断言式
- **典型表达**：
  > "当他手握刀剑的时候，就不再是你认识的那个少年了。"
  > "如果艾晴不能活，他就绝对不会后退。"
- **对话特征**：从不说废话，每句都是判断/结论

### 阴崖
- **口吻**：温和恳切、内藏无力、欲言又止
- **典型表达**：
  > "这是哥哥唯一能为你做的事情了。"
  > "今晚就走。"
  > 叹息、犹豫、无声离去
- **对话特征**：话越少，越说明事态严重

## 对话中的情感节奏

### 1. 沉默>语言
本作中最有力的"对话"往往是沉默：
> 「阴崖再没有说话。时光静静地流逝。」
> 「在沉默中，末三没有再说话。诸红尘也没有。」
→ 沉默的段落越长，情感重量越大

### 2. 冷面升温
表面冷淡的对话下情感逐步升温：
> 槐诗的冷面杀人→乌鸦兴奋评价→kp惊叹→幕后人赞许
→ 角色不表达感情，但周围人的反应为他增温

### 3. 锋利自嘲
角色用自我攻击的方式表达真实伤痛：
> 艾晴反复说"女疯子""瘸子"——越用力贬低自己，越说明创伤之深

### 4. 朴素收尾
激烈情感后以朴素日常结束：
> 训话结束→「该干什么干什么去吧」
> 恳请失败→阴崖无声离去→艾晴拿起水果继续工作
→ 日常细节消化情绪，比直接抒情更有力
