# 节奏风格 (Rhythm Style)

> 记录《天启预报》的段落结构、章节节奏和信息密度分布。

## 段落长度分布

| 类型 | 占比 | 使用场景 |
|------|------|----------|
| **短段**（1-2句） | ~40% | 战斗动作、情绪留白、deadpan吐槽、环境断句 |
| **中段**（3-5句） | ~35% | 日常叙事、对话交锋、设定说明、角色互动 |
| **长段**（6句以上） | ~25% | 环境铺排、家族训话、战斗大场面、角色背景揭示 |

### 与术师手册的差异
术师手册60%为短段，天启预报短段占比降至40%左右。长段占比显著增加（10%→25%），尤其在环境描写、战场铺排和家族训话场景中，作者愿意用较长的段落营造沉浸氛围。

### 短段的使用特征
- **诗意断句**：一句一段制造留白与节奏停顿，是核心节奏技法
  > 「低下头。」
  > 「只是低沉。」
  > 「好像大提琴残留的弦音。」
- **战斗动作帧**：短句快切模拟电影画面分镜
  > 「白马向前，飞奔，发出震怒的嘶鸣。」
  > 「槐诗咆哮。」
- **冷幽默独白**：以极短段落投放黑色幽默
  > 「就这？」
  > 「了账。」

### 长段的使用时机
- 末世级环境描写（珠穆朗玛开篇、巨人钓鱼场景）
- 家族训话/长辈演讲（林家老人的训词）
- 独骑冲阵的全景叙述（赏金猎人防线场景）
- 角色背景回溯（艾晴与阴崖的对话）
- 组织运作说明（亚洲新秀赛参赛情况）

## 章节结构

- **章节字数**：约2000-4000字/章，比术师手册长约2-3倍
- **章节标题**：数字序号+简短标题，多为对象或事件概括
  > 「第184章 惨败」
  > 「第260章 舍不得孩子套不着狼」
  > 「第287章 王子与公主」
  > 「第311章 最后的问题」
- **章末处理**：
  - 诗意留白收尾（约40%）
  - 悬念钩子断章（约25%）
  - 作者旁白附注（约20%）
  - 冷幽默反转结尾（约15%）
- **章内分隔**：使用 `.` 或 `……` 标记场景切换/时间跳跃

## 核心节奏模式

### 1. 暴力-静寂节奏（标志性）
```
紧张铺垫 → 暴力爆发（快切短句）→ 骤然静寂 → 诗意留白 → 环境声音 → 新段落
```
示例（直播间杀戮后）：
1. 枪声轰鸣+骨肉碎裂（暴力高潮，短句急促）
2. 「很快，一切都恢复了静寂。」（骤停）
3. 椅子拖曳声、没入血泊（声音缓降）
4. 「好像大提琴残留的弦音。」（诗意落点）
5. 槐诗坐下，看摄像头（新节奏启动）

### 2. 日常-暗流节奏
```
轻松日常 → 暗示性细节 → 表面平静 → 突然转折/暗流涌动
```
示例（艾晴线）：
1. 阴崖来访，送水果（日常）
2. 讨论老太爷道歉（暗流）
3. 放下机票、告别（暗示）
4. 「是时星夜晴朗。万里无云。」（反讽收尾，暗示被拦截）

### 3. 递进冲锋节奏（独骑叙事）
```
远距离叙述 → 逐步逼近 → 节奏加速 → 对峙 → 冲锋爆发
```
示例（槐诗冲阵）：
1. 诸红尘和末三远程分析（远距离）
2. 名单上名字不断出现（中距离，递进）
3. 「还有十公里……」（逼近计数）
4. 马蹄声渐近（声音递进）
5. 骑士出现→对峙→冲锋（爆发）

### 4. TRPG段落节奏
```
模组设定 → 角色行动 → kp反应/判定 → 意外展开 → 回到现实
```
带有TRPG跑团的松散交互节奏，角色行动与kp判定交替推进。

### 5. 家族训话节奏
```
长辈开口（威严）→ 核心训诫（递进排比）→ 提问 → 回答 → 总结 → 展望/散场
```
林家老人训话的节奏是演讲式的——先立规矩，再用反问推高情绪，最后温情收尾。

## 场景切换方式

### 1. `.` 场景分隔符
最常用的章内切换方式，以 `.` 独占一行标记：
> 段落结束 → `.` → 新场景/新时间

### 2. 时间/环境锚定
新段落以时间或环境变化开始：
> 「翌日，清晨。」
> 「傍晚的时候，淅淅沥沥地下起了小雨。」
> 「日子就这样一天天的过去，波澜不惊，风平浪静。」

### 3. 视角切换
通过新视角角色的感知或对话开始：
> 「'可以啊，小子。'那个粗豪的男人用力地拍了一下他的肩膀——」（切至老肖视角场景）

## 信息密度分布

| 场景类型 | 信息密度 | 特征 |
|----------|----------|------|
| 环境/氛围描写 | 低 | 大量留白和诗意渲染 |
| 日常搞笑 | 中-低 | 角色互动和吐槽为主 |
| 组织运作/赛事 | 高 | 多组织多角色信息密集 |
| 家族训话 | 中-高 | 世界观嵌入训话 |
| 战斗场景 | 中 | 动作+诗意描写+角色反应交叉 |
| 角色背景回溯 | 中 | 对话驱动，细节丰富 |
| 独骑冲阵 | 中-低 | 反复递进，氛围 > 信息 |
| TRPG模组段 | 中-高 | 叙事与游戏机制交织 |

## 节奏加速技巧

1. **连续短句成段**模拟电影快切
2. **声音密集化**（蹄铁声从稀疏到密集、枪声连响）
3. **视角快切**（敌方恐惧→主角沉默→旁观者分析）
4. **删除对话标签**保持对话速度
5. **递减距离计数**（「还有十公里……」→「最后的一公里！」）

## 节奏减速技巧

1. **诗意环境铺排**（雨、星、雾、废墟）
2. **角色沉默**以单字段落标记：「……」
3. **座钟/时间标记**插入：「时光静静地流逝。当座钟的声响起的时候——」
4. **反讽式平静**：在紧张事件后以看似无关的平静叙述制造反差
5. **食物/料理细节**：通过烹饪细节减速，同时展示世界观深度
