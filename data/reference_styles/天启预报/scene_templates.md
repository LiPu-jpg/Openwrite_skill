# 场景模板 (Scene Templates)

> 记录《天启预报》中反复出现的场景结构模式。

## 模板1：暴力美学战斗场景

### 结构
```
环境铺垫（诗意渲染）→ 对峙/寂静 → 暴力爆发（短句急促）→ 骤然静寂 → 诗意留白 → 战后余韵
```

### 特征
- 战斗前必有**环境氛围营造**（雨、雷、灯光、废墟）
- 暴力描写以**通感美学**呈现（血→鸢尾花、暴力声→弦音）
- 战斗过程使用**电影分镜式短句**，极速推进
- 暴力高潮后必接**骤然静寂**——声音从满到空的转折
- 战后以**诗意段落**或**冷幽默**收尾

### 示例结构（直播间杀戮·第183章前后）
1. 昏暗灯光+地下室环境渲染
2. 槐诗与剩余敌人对峙：「就这？没别的了？」
3. 枪声+骨肉碎裂+斧刃劈砍（快切短句）
4. 「很快，一切都恢复了静寂。」
5. 椅子声没入血泊→「好像大提琴残留的弦音」
6. 槐诗坐下看摄像头→竖中指→直播结束

## 模板2：独骑冲阵场景

### 结构
```
远方观察/分析 → 递进逼近（声音+距离递减）→ 敌方心理崩溃 → 对峙呼喊 → 无声冲锋 → 爆发
```

### 特征
- 以**旁观者视角**开始，从远处逐步接近主角
- **声音递进**是核心节奏工具（蹄铁声从稀疏到密集）
- 敌方的**恐惧心理**是重要叙事内容
- 主角保持**沉默**——越沉默越有威慑力
- 对峙中敌方理性劝说→主角无回应→冲锋

### 示例结构（石城入口冲阵·第311章前后）
1. 诸红尘+末三远程分析槐诗心理
2. 赏金猎人紧急建立防线（旁观者视角展示恐惧）
3. 骑士骑白马从隧道缓缓走出——遍体鳞伤+滴血枪刃+脚下鸢尾
4. 指挥者呼喊劝降→槐诗无回应
5. 蹄铁声变密→雷鸣→冲锋→对峙粉碎
6. 「——最后的一公里！」

## 模板3：家族/组织训话场景

### 结构
```
长辈登场（威严） → 排比训诫（递进式）→ 反问 → 回应 → 温情/放行 → 后续安排
```

### 特征
- **演讲式独白**：长辈连续多段输出，中间偶有晚辈回应
- **排比递进**：同一句式反复推高情绪（「去拼，去抢，去夺！」）
- **残酷温情并置**：「死了也是白死」→「你们都有远大的前程」
- 训话结束以**轻描淡写收尾**：「该干什么干什么去吧」

### 示例结构（林家训话·第259章前后）
1. 老人端坐，目如日轮
2. 训话：路的选择+林家规矩+生死态度
3. 「如果你们死在外面的话，该怎么办呢？」
4. 少年回答：「就是白死了。」
5. 老人赞许→温情收尾→个别关照（孙媳妇处理）
6. 老人闭眼沉睡→天地昏暗

## 模板4：角色背景/家族恩怨场景

### 结构
```
来访铺垫 → 表面寒暄 → 核心试探/恳请 → 冰山对话 → 锋利拒绝 → 无言离去 → 暗示性收尾
```

### 特征
- 对话是**冰山式**——水面下的信息远大于水面上
- **沉默**是最有力的台词
- 离去后以**物件/环境**做隐喻收尾（机票留在原地、清洁工扫走）
- 事件真相通过**后续细节**间接揭示（当晚只有去伦敦的航班能飞→万里无云）

### 示例结构（艾晴与阴崖）
1. 阴崖来访，带水果（日常伪装）
2. 传达老太爷道歉→艾晴冷静反驳
3. 恳请回家→艾晴以"女疯子"反击
4. 阴崖放下机票：「想办法调到国外去吧——今晚就走。」
5. 艾晴拒绝→阴崖无声叹息离去
6. 艾晴拿水果回日程→机票被清洁工扫走
7. 「是时星夜晴朗。万里无云。」（反讽暗示）

## 模板5：宏大超自然事件场景

### 结构
```
环境异象（诗意描写）→ 超自然力量降临 → 行动指令 → 团队冲锋 → 混战 → 叙述回落
```

### 特征
- 开场以**奇幻视觉**铺开——巨人幻影、钓钩降临、飓光龙卷
- **规模感**通过环境+声音传递，而非数字罗列
- 行动中穿插**组织化指挥**（领队命令、社保局出击）
- 战斗中信息密度升高（阵阵触发、多线并行）

### 示例结构（巨人钓鱼/大楼突入）
1. 「傍晚的时候，淅淅沥沥地下起了小雨」→ 雨中幻影
2. 龙伯之国的巨人甩出钓钩→大楼被拽入独立边境
3. 领队响指→飓光撕裂大楼表层→通路打开
4. 「快快快！！！」→升华者冲锋→陷阱触发→轰鸣不断

## 模板6：TRPG/模组场景

### 结构
```
模组启动（kp介绍）→ 角色行动 → 判定/意外 → 角色发挥 → kp评价 → 回到现实 → 清算
```

### 特征
- 将叙事嵌套在**TRPG框架**中（有kp、有骰子、有模组结束语）
- 角色在模组中的表现被**考官评价**
- 模组与现实的边界**模糊**——模组中的风险是真实的
- 回到现实后的**认知错位**产生喜剧效果（槐诗不知道自己在被考试）

### 示例结构（转正考察模组）
1. 槐诗在模组中行动→乌鸦评价→kp记录
2. 模组结束→「架空结局：无'人'生还」
3. 乌鸦向主持者陈述→kp惊叹「妙哉！」
4. 槐诗醒来→一脸懵→老肖拍肩「表现不赖」
5. 老监察官解释这是转正考试→槐诗傻眼→艾晴也上当了

## 模板7：竞技/赛事场景

### 结构
```
赛事铺垫（组织/规模说明）→ 选手登场 → 召集（钟声/聚光灯）→ 仪式化对峙 → 挑战宣言 → 战斗
```

### 特征
- 大量**世界观信息**通过赛事参赛者和组织说明自然嵌入
- 仪式感强（钟声、聚光灯、观众席、直播间）
- **王子vs骑士**的中世纪叙事模式
- 角色间的对峙以**尊重+兴奋**为基调，而非仇恨

### 示例结构（亚洲新秀赛·乐园段落）
1. 赛事运作和多组织参赛说明
2. 钟声响起→升华者被传送到剧场
3. 王子骑白马登场→宣言：「让我来分辨，你有没有继承这份尊荣的资格！」
4. 槐诗的血液沸腾→兴奋→踏步上前
5. 「那么，王子殿下，遵照乐园的规则，我在此向你发起挑战！」

## 模板8：烹饪/生活细节场景

### 结构
```
食物/技艺展示 → 世界观嵌入（烹饪=超自然能力）→ 人情交往 → 名片/善缘 → 温暖收尾
```

### 特征
- 烹饪细节本身即世界观展示（天狗山秘传焚合、黑暗料理、深渊药材）
- 食物场景是**节奏减速区**——让读者和角色一起喘口气
- 人际关系通过**物件交换**（名片、便当）自然建立
- 收尾温暖但不煽情

### 示例结构（渡边主厨送别）
1. 渡边展示焚合料理的神奇效果
2. 暗示天狗山的更广阔世界→槐诗婉拒深入
3. 递名片+料理便当→善缘建立
4. 「渡边露出一丝笑容，送槐诗走出三步」→日常温暖

## 模板9：异境/永世之战场景

### 结构
```
陌生环境探索 → 信仰者遭遇 → 身份错位（被当作神/大灵）→ 荒诞对话 → 世界观揭示 → 选择
```

### 特征
- 主角被误认为**救世主/大灵**，产生荒诞式身份错位
- 信仰者的虔诚与主角的困惑形成**反差喜剧**
- 异境有独立的文明体系和末世危机
- 选择往往是**被迫卷入**而非主动参与

### 示例结构（永世之战·大灵）
1. 僧侣虔诚磕头→槐诗困惑
2. 大灵=我？「……那你们国家岂不是很可怜？」
3. 僧侣解释永世之战→国家衰败→唯有大灵能救
4. 槐诗内心：「总感觉去了是做工具狗」
5. 傅依提供情报→天文会内部讨论组出现→选择是否卷入

## 模板10：沉默对峙场景

### 结构
```
双方在场 → 一方开口/行动 → 另一方沉默 → 沉默拉长 → 沉默打破（行动/离去） → 余韵
```

### 特征
- **沉默本身就是对话**——不需要语言
- 沉默的长度与情感重量成正比
- 打破沉默的方式揭示角色选择（离去=放弃、行动=决心）
- 余韵通常以环境/物件呈现

### 示例
> 「阴崖再没有说话。时光静静地流逝。当座钟的声响起的时候，他起身——」
> 「在沉默中，末三没有再说话。诸红尘也没有。只有名单上，一个个名字不断的出现——」
