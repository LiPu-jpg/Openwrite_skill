## 核心角色约束

### 艾华斯·莫里亚蒂
| 约束项 | 规则 |
|--------|------|
| 身份三重性 | ①穿越玩家 ②莫里亚蒂家族成员 ③恶魔学者 — 三层身份不可混淆 |
| 游戏知识边界 | 知道"游戏版本"内的剧情，但"这一世"已有偏差，不能全知 |
| 称呼习惯 | 对外温和礼貌、对内游戏吐槽；不用粗俗语言 |
| 战斗风格 | 恶魔学者（召唤/契约）+ 牧师（治愈/祝福）双修 |
| 情感约束 | 有温度但保持距离；不主动表白、不轻易信任 |
| 叙述限制 | 游戏术语（BOSS/NPC/版本号）只出现在内心独白，不说出口 |
| 决策模式 | 提前布局型：总是为"未来的版本"做准备 |

### 哈伊娜
| 约束项 | 规则 |
|--------|------|
| 职业身份 | 见习监察→正式监察，必须按规程行事（虽然常打破） |
| 性格核心 | 紧张冒失但善良正直；不能变成"冷酷精英" |
| 与艾华斯关系 | 调查者与被调查者的暧昧关系；始终保有职业本能 |
| 社交模式 | 初见紧张、职业性提问是条件反射、成长后仍保留"毛糙"基底 |
| 战斗能力 | 实力在成长中，不能过早变强；每次进步要有明确理由 |

### 伊莎贝尔
| 约束项 | 规则 |
|--------|------|
| 成长轨迹 | 社恐→沉默→开口→自信→领导者，不可跳步 |
| 身份约束 | 公主身份带来的政治责任不可回避 |
| 对话量 | 与自信程度正相关，早期极少开口 |
| 与艾华斯关系 | 政治同盟+微妙情感；不能过早表态 |
| 王族素养 | 即使社恐，也有基本的贵族修养和判断力 |

### 奥斯瓦尔德
| 约束项 | 规则 |
|--------|------|
| 精灵特征 | 长寿（数百年）、骄傲、拉长音的口音 |
| 忠诚指向 | 忠于莫里亚蒂家族，非忠于个人 |
| 讽刺守则 | 越讽刺=越关心；真正不在乎的人直接"请出去" |
| 能力水平 | 极强但极少出手；出手=情况极端 |
| 与艾华斯关系 | 管家↔少爷，但实际上是亦师亦友 |

### 夏洛克
| 约束项 | 规则 |
|--------|------|
| 智力水平 | 极高，但展现方式是"不动声色" |
| 掩饰习惯 | 永远淡化自己的发现（"大概""碰巧"） |
| 与艾华斯关系 | 默契信任；是艾华斯为数不多可以放松的人 |
| 对话风格 | 话少、精准、永远有弦外之音 |

### 莉莉
| 约束项 | 规则 |
|--------|------|
| 存在感 | 影子般；出现时不做声，做完事就消失 |
| 战斗本能 | 黄昏之道使者，暗杀能力极强 |
| 情感表达 | 通过行动而非语言（递茶=关心、出现在身后=保护） |
| 称呼 | 叫伊莎贝尔"大小姐" |

## 世界规则体系

### 超凡规则
| 规则 | 内容 |
|------|------|
| 能级压制 | 高能级对低能级有绝对优势（但不绝对碾压） |
| 道途排他 | 一人通常只走一条道途，多道途极为罕见（艾华斯是例外） |
| 进阶条件 | 每次进阶需要特定材料、仪式、精神条件 |
| 恶魔契约 | 召唤恶魔必须付出代价，代价与力量成正比 |
| 梦界规则 | 梦界中思维=力量，精神力决定一切 |
| 圣痕暴露 | 使用超凡力量可能产生圣痕，暴露身份 |
| 牧养限制 | 牧养恶魔需要持续供能，不能无限制养 |

### 社会规则
| 规则 | 内容 |
|------|------|
| 超凡保密 | 普通人不完全知道超凡世界的存在（有限公开） |
| 贵族法则 | 血统和家族决定社会地位；安可变但缓慢 |
| 监察权限 | 监察局有调查权但受限于政治干预 |
| 国际条约 | 各国间有外交协议，超凡战争受规则约束 |
| 教会影响 | 宗教对道途认知有重大影响（某些道途被视为异端） |

### 游戏-现实对应规则（仅艾华斯视角）
| 游戏概念 | 现实对应 | 约束 |
|----------|----------|------|
| 等级/经验值 | 力量积累/战斗经验 | 现实中积累远慢于游戏 |
| BOSS | 高能级敌人 | 现实中不遵循游戏阶段设计 |
| NPC | 这个世界的真实人物 | 他们有自由意志，不可预测 |
| 剧情线 | 历史事件 | 已产生偏差，不能完全依赖 |
| 版本号 | 大规模灾变节点 | 时间线可能提前或推迟 |
| DLC | 支线历史事件 | 可能已被改变 |
| 攻略 | 过去玩家的经验总结 | 在偏差后可能失效 |

## 语言/格式约束

| 约束 | 说明 |
|------|------|
| 对话引号 | 全文使用「」直角引号，不使用""双引号 |
| 用字习惯 | 使用「幺」而非「么」（如「什幺」不是「什么」） |
| 称谓规范 | 正式场合：姓+先生/小姐；亲近：直呼名；王族：殿下/陛下 |
| 游戏术语范围 | BOSS/NPC/DLC/版本号/经验值/抽卡/进阶——仅限艾华斯内心 |
| 作者注格式 | 正文结束后分隔线，口语化，1-5段 |
| 多语种标注 | 跨国场景需标明说的什幺语言 |
| 英伦措辞 | 维持维多利亚风措辞（"有所冒犯"、"还请见谅"、"若非"） |

## 写作禁忌清单

### 绝不可做
1. **不让艾华斯对外说出游戏术语**（"版本号""BOSS"只在内心）
2. **不让伊莎贝尔跳过成长步骤**（突然变话多/自信必须有铺垫）
3. **不让奥斯瓦尔德失去优雅**（即使愤怒也是冷的）
4. **不让哈伊娜变成"完美精英"**（紧张冒失是核心魅力）
5. **不写无代价的超凡使用**（每次都要消耗/后果）
6. **不做游戏知识的完全预知**（"这一世"已有偏差）
7. **不让反派扁平化**（每个反派有自己的逻辑和尊严）
8. **不写过度血腥/猎奇**（暴力描写克制有度）
9. **不让作者注进入正文**（两个语域严格分离）
10. **不在正文用""双引号**（全文「」）

### 慎重操作
11. **穿越者身份揭露** — 只能渐进且有重大故事后果
12. **多道途同修** — 艾华斯是特例，不可轻易给其他角色
13. **国际关系变动** — 需要铺垫期，不可突然翻转
14. **角色死亡** — 本作基调温暖，重要角色死亡需极其慎重
15. **能级跳跃** — 每次升级必须有充分铺垫和代价
16. **恋爱关系推进** — 不催、不急，让关系自然发展
17. **精灵相关设定** — 精灵是超长寿种族，相关历史跨度极大
18. **梦界规则** — 梦界物理法则灵活但需内部一致

## 与其他作品的一致性差异

| 维度 | 牧者密续 | 天启预报 | 术师手册 |
|------|----------|----------|----------|
| 对话引号 | 「」 | "" | "" |

| 主角类型 | 全知预谋型 | 孤独奋战型 | 成长印证型 |
| 女性角色密度 | 高且各有弧光 | 中等 | 较低 |
| 政治比重 | 极高（核心线之一） | 低 | 极低 |
| 幽默类型 | 反差萌/社恐/游戏吐槽 | 黑色/冷面 | 轻松/日常 |
| 作者介入 | 极高（章末作者注~80%） | 中（~50%） | 极低 |
