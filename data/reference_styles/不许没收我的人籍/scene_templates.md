# 场景模板 (Scene Templates)

> 记录《不许没收我的人籍》中反复出现的场景结构模式。

## 模板1：话术说服场景

### 结构
```
目标人物登场（带戒备/敌意）→ 兰奇观察分析 → 建立信任/共情 → 逐步引导 → 击中核心 → 对方动摇 → 达成目的
```

### 特征
- 战斗的核心不是武力而是**语言**——兰奇的"话疗"是最强武器
- 过程中穿插**对方内心视角**的切换，展示被说服的过程
- 兰奇的话术看似温和但**逻辑封闭**，给出的选择实质上只有一个最优解
- 说服成功后往往有**轻幽默收尾**（对方自嘲或旁人吐槽）

### 示例结构（说服普拉奈·第449章前后）
1. 普拉奈登场：大魔族，戒备心极强，不信任人类
2. 兰奇以"大爱诗人"身份入场：文化认同而非武力威胁
3. 提及安塔纳斯/辛诺拉已归顺：事实论证
4. 分析北大陆魔族当前困境：共情+理性分析
5. 提出合作愿景：给出"可以拒绝"的选择（实际无法拒绝）
6. 普拉奈沉默良久→同意
7. 轻幽默结尾

## 模板2：影世界副本场景

### 结构
```
副本信息铺垫 → 团队集结 → 进入影世界 → 初期探索 → 遭遇难题/敌人 → 兰奇策略解题 → 战斗高潮 → 通关评价
```

### 特征
- 类MMORPG/TRPG的副本探索结构
- 兰奇的**策略解题**是重点——不靠蛮力而靠头脑
- 团队配合场景展示各角色的技能特色
- 通关后有数据化的**评价/排名**（挑战者位阶积分等）

### 示例结构（入学考试影世界·第18-19章）
1. 考试规则说明：影世界通关，有时间限制
2. 兰奇进入影世界：观察环境、判断机制
3. 快速通关（9分钟，远超平均45分钟）
4. 弗雷7秒通关：天才对比
5. 系统终端被兰奇搞崩：意外事故
6. 监考评价：惊讶+关注

## 模板3：修罗场名场面

### 结构
```
日常中的伏笔 → 意外发现/巧合 → 二人/三人对峙 → 尴尬升级 → 外部打断 → 收场但余波未平
```

### 特征
- 感情线的核心场景类型
- **日系轻小说式修罗场**：不是阴暗的情感纠葛，而是带喜感的尴尬
- 多层误会叠加制造笑料
- 总在最尴尬的时刻被**第三方/第四方撞见**
- 事后感情关系会**微妙推进**而非后退

### 示例结构（肖像画事件·第495-497章）
1. 塔莉娅因某事进入兰奇房间（日常）
2. 发现墙上挂着她的肖像画——震惊
3. 兰奇试图解释→塔莉娅红着脸质问
4. 休柏莉安闻声赶到→用身体护住兰奇
5. 三人僵持→家人/管家闯入
6. 全场冻结→尴尬收场
7. 后续：三人关系更近但都假装无事发生

## 模板4：格斗赛/竞技场景

### 结构
```
赛前准备/训练 → 对手分析 → 登场+全场氛围 → 前期交锋 → 陷入劣势 → 策略翻盘 → 胜利+赛后评价
```

### 特征
- 标准热血少年漫的竞技套路
- **训练蒙太奇**在赛前占重要篇幅
- 兰奇即使不亲自上场，也以**教练/策略师**身份主导
- 格斗家被改造/训练后的**实力差展示**是爽点
- 赛后评价往往带出更大格局（学院排名、势力关注）

### 示例结构（格斗赛·第221-223章）
1. 兰奇"军校式"特训格斗家——蒙太奇
2. 库塞（亚洛兰反派）登场：实力强劲+气焰嚣张
3. 竞技场开战：格斗家vs敌方
4. 初期看似势均力敌→敌方使诈
5. 休柏莉安出手保护兰奇（场外冲突）
6. 格斗家展示训练成果→翻盘
7. 赛后评价+兰奇的声誉提升

## 模板5：间谍/卧底日常

### 结构
```
伪装身份的日常（教学/社交）→ 可疑人物接近 → 试探与反试探 → 危机冒头 → 化解/推迟 → 继续伪装
```

### 特征
- 帝国间谍线特有的场景模式
- 「兰德里教授」的**学术日常**与**间谍暗战**平行
- **红隼尼古拉**等帝国情报人员的试探构成持续紧张感
- 塔莉娅通过**心念**远程支援增加多线联动
- 身份暴露的**悬念**贯穿整条线

### 示例结构（帝国线·第727-728章）
1. 兰奇以"兰德里教授"身份在圣克瑞瑅修道院讲课
2. 学生们被他的魔法工学讲座吸引
3. 红隼尼古拉（帝国特别行动处）来访：试探性邀请
4. 兰奇假装犹豫→接受"名誉顾问"身份
5. 塔莉娅通过心念传来情报
6. 表面平静地继续教学→内心进行多线策略推演

## 模板6：团聚/日常聚会

### 结构
```
场景设定（猫老板餐厅/学院等）→ 角色陆续到场 → 闲聊/吐槽 → 提及某个梗（鸽子/消防安全等）→ 温馨互动 → 偶有新信息引出 → 温暖收尾
```

### 特征
- 本书最高频的场景类型——温馨日常是基底
- 大量角色群像互动：每个角色有自己的吐槽位和角色感
- 通过**固定梗**制造亲切感和重复笑料
- 偶尔在日常中**嵌入伏笔**或引出下一个事件

### 示例结构（猫老板餐厅聚会·第877-878章）
1. 猫老板餐厅：普拉奈、西格丽德、猫老板等多人在场
2. 闲聊歌剧公演准备→安塔纳斯的广播节目
3. 情人节的氛围→修罗场暗流
4. 角色间的调侃和吐槽
5. 温馨收尾：大家庭的感觉

## 模板7：管家/仆从暗中行动

### 结构
```
兰奇吩咐/暗示 → 管家汉斯微笑领命 → 跳转至结果 → 问题已被悄然解决
```

### 特征
- 管家汉斯的"幕后处理"是固定叙事技巧
- 过程往往被**省略**，只展示结果——制造管家的"无所不能"感
- 管家的完美笑容本身就是一种修辞：不透露任何手段
