# 对话风格 (Dialogue Style)

> 记录《不许没收我的人籍》的对话格式、角色语言个性、对话驱动模式和互动风格。

## 对话格式规范

### 基本格式
- **所有对话**统一使用「」直角引号（全书最显著格式特征）
  > 「兰奇先生，请问您能解释一下这是怎么回事吗？」
  > 「本小姐才没有在意那种事。」
- **内心独白**：无引号，直接融入叙述或以括号/破折号标示
  > 兰奇在心里默默给这个局面打了个三星评价——不算太糟，但也绝对算不上好。
- **心念通讯**：塔莉娅与兰奇的心灵感应通讯以叙述描写说明
  > 塔莉娅通过心念向兰奇传达了警告——前方有埋伏。

### 对话标签使用
- **动作标签为主**：以角色的表情/动作/姿态作为对话附带描写
  > 塔莉娅双手抱胸，目光微微偏移：「我只是碰巧路过而已。」
  > 管家汉斯微微躬身，露出标准的45度微笑：「少爷，一切已安排妥当。」
- **"说道"类标签较少**：叙述偏向用行为描写替代"说道""问道"
- **省略标签**在快速对话中频繁出现：多人对话时靠角色语气特征区分说话者

### 格式分配规则
| 对话类型 | 格式 |
|----------|------|
| 角色口头对话 | 「」 |
| 内心独白 | 无引号融入叙述 |
| 心念通讯 | 叙述描写说明 |
| 游戏系统提示 | 视情况用特殊排版 |
| 作者附注 | ps：文本 |

## 对话驱动模式

### 1. 话术型长独白（核心模式）
兰奇说服他人时的长篇话术是本书最有辨识度的对话形式：
```
「你有没有想过，为什么北大陆的魔族几千年来始终无法统一？
不是因为力量不够，而是因为没有人愿意第一个放下仇恨。
我来这里不是要征服谁——我只是想让每一个人都有选择的权力。
你可以选择相信我，也可以选择离开——但如果你离开了，那你要面对的不是我，而是那些已经相信了我的人。」
```
特征：
- 先共情→再分析→给出选择
- 看似温和但逻辑压迫感极强
- 语气始终平和，不使用威胁性语言
- 经常以反问或开放式选择结束

### 2. 修罗场三角对话
三个（或更多）角色同时在场时的感情张力对话：
```
塔莉娅：「兰奇，你能解释一下为什么你的房间里会有本小姐的肖像画吗？」
兰奇：「呃……那个，是学院的作业……」
休柏莉安（挡在兰奇前面）：「塔莉娅大人，兰奇先生需要休息，请您不要为难他。」
塔莉娅：「……你？」
```
特征：
- 三方各有立场和潜台词
- 对话中穿插角色表情/动作的快速描写
- 喜剧感与感情张力并存
- 经常被外部打断（家人闯入、管家敲门等）

### 3. 日常调侃对话
温暖的日常互动——吐槽、打趣、轻松：
```
猫老板：「今天的特餐是烤鱼——不是鸽子，放心。」
兰奇：「……猫老板你这话是对谁说的。」
塔莉娅（磨牙声）：「……」
弗雷：「嘿，那个鸽子事件到底是怎么回事？每次提起来……」
兰奇：「不要提了。」
```
特征：
- 依靠角色间的固定梗（鸽子事件、消防安全等）制造笑点
- 对话节奏轻快，换行频率高
- 多人同时参与，气氛热闹

### 4. 师生/训练式对话
兰奇教导格斗家或学员时的训练对话：
```
「你的第三击总是比第二击慢0.3秒——这在实战中是致命的。
再来一次，这次注意你的重心转移。
记住，格斗不只是力量——是节奏。」
```
特征：
- 专业性+实用性
- 简洁直接，无废话
- 与兰奇日常的温和形成反差（教学时格外严格）

### 5. 间谍伪装对话
在帝国间谍线中，兰奇以"兰德里教授"身份说话：
```
「各位学员，今天我们要讨论的是魔法工学第三定律——关于能量守恒在法阵构建中的应用。」
```
特征：
- 学术化的正式用语
- 教授身份与间谍身份的双层对话
- 暗语和双关：表面在讲学，实际在传递情报

## 对话与叙事的交织方式

本书的对话从不独立于叙事——每段对话都伴随着：
1. **表情/动作描写**：在对话行间穿插角色的微表情和小动作
2. **内心补充**：对话后跟随角色的内心真实想法（与口头表达形成反差）
3. **环境反应**：其他在场角色的即时反应构成多层信息传递

示例：
> 「兰奇先生，我有件事想跟你谈谈。」塔莉娅的语气看似平淡，但她的手指正在无意识地搓着袖口——这是她紧张时的习惯性动作。
> 兰奇注意到了这一点，心里迅速评估了几种可能的话题。
> 「当然，塔莉娅小姐，请说。」他露出一个恰到好处的微笑。
