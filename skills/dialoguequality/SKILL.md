# 对话质量系统

提取和分析角色对话风格，确保对话自然、个性化，识别AI味对话问题。

## 核心概念

**对话指纹**包括：
- `avg_sentence_length` - 平均句长
- `common_bigrams` - 常用词组
- `question_ratio` - 问句比例
- `speech_patterns` - 口头禅/语言习惯

## 可用工具

| 工具 | 说明 |
|------|------|
| `extract_dialogue_fingerprint` | 提取对话风格指纹 |

## 使用示例

### 分析角色对话风格

```
用户: 分析一下张三的对话风格

AI:
[COMMAND] extract_dialogue_fingerprint {
  "chapter_id": "ch_005",
  "character_names": ["张三"]
}

结果:
- 角色: 张三
- 平均句长: 12.3字
- 常用词组: ["脸色一沉", "沉声道", "眉头一皱"]
- 问句比例: 15%
- 口头禅: ["你懂什么", "且慢"]
```

### 对话AI味检测

```
用户: 这段对话有没有AI味？

AI:
检测结果:
⚠️ 过度礼貌：3处"请"、2处"劳烦"
⚠️ 解释性对话：角色在互相解释已知的背景信息
⚠️ 完美逻辑：每句话都有直接因果
⚠️ 情绪单一：缺少对话中的微妙情绪波动

建议:
1. 减少"请"的使用，对话中普通人不会这么客气
2. 删除角色A向角色B解释"B应该知道的事"的内容
3. 加入角色口吃、停顿、或者话说一半
```

## 一、对话风格要素

### 1. 句长
| 角色类型 | 建议句长 | 示例 |
|---------|----------|------|
| 武将/莽汉 | 短句（3-8字） | "滚！""拿命来！""你算什么东西！" |
| 文人/谋士 | 长句（15-30字）| "此事还需从长计议..." |
| 普通角色 | 中等（8-15字）| "你说的有道理，但..." |

### 2. 口头禅
- 角色特有的习惯用词
- 情绪触发词（如愤怒时的口头禅）
- 身份标志词（如古人的"哉"、"乎"）

### 3. 问句比例
| 角色类型 | 问句比例 |
|---------|----------|
| 好奇角色 | >20% |
| 普通角色 | 5-15% |
| 果断角色 | <5% |

### 4. 语言特征
- 方言/口音
- 专业术语（修士说"灵力"而非"能量"）
- 情绪词频率

## 二、AI味对话检测清单

### 2.1 过度礼貌（warning）
**问题特征**：
- "请"、"劳烦"、"久仰"、"阁下"泛滥
- 普通角色之间对话像外交辞令

**改进方向**：
- 普通人对话少用"请"
- 根据角色身份调整礼貌程度
- 加入不耐烦、打断、敷衍

### 2.2 解释性对话（warning）
**问题特征**：
- 角色A向角色B解释B应该知道的事
- 用于向读者传达信息，而非角色间交流

**典型模式**：
```
❌ "你应该知道，我们修仙之人最重心性"
❌ "正如我之前所说，这件事..."
```

**改进方向**：
- 删除一方对另一方解释对方应知的内容
- 用行动或沉默代替解释

### 2.3 完美逻辑（warning）
**问题特征**：
- 每句话都有直接因果
- 没有停顿、思考、犹豫
- 对话像辩论赛，有问必有答

**改进方向**：
- 加入角色答非所问
- 加入沉默、或话题转移
- 让角色话说一半被打断

### 2.4 情绪直白（warning）
**问题特征**：
- 对话中直接说出情绪状态
- "我很生气！""我恨你！"

**改进方向**：
- 用语气词、停顿、动作代替情绪词
- 让情绪通过对话节奏体现

### 2.5 套路对话（warning）
**检测模式**：
- "受教了" - 学到东西后必说
- "道友所言甚是" - 修仙文标配
- "X兄所言极是" - 过度的认同
- "依你便是" - 毫无挣扎的顺从

**改进方向**：
- 删掉这些套路回应
- 用更有角色特色的反应替代

### 2.6 缺乏个人特质（warning）
**问题特征**：
- 所有角色说话方式差不多
- 知识分子和文盲用词相同
- 古人和现代人说话没区别

**改进方向**：
- 根据角色背景设计专属词汇
- 用方言、口音、专业术语区分

## 三、对话改进示例

### 示例1：过度礼貌

```
❌ AI版本:
"张道友，请留步。方才之事，实非在下本意，若有得罪之处，还望海涵。"
"李兄，何必如此客气？有话直说便是。"

✅ 人味版本:
"老张！"（追上去）
"干嘛？"
"方才的事……"（挠头）
"行了行了，别说了。"

解释: 普通修士不会这么文绉绉，尤其是熟人之间。
```

### 示例2：解释性对话

```
❌ AI版本:
"既然我们都是青云宗的弟子，就应该互相扶持..."
（实际上对方也是青云宗的，不需要提醒）

✅ 人味版本:
（沉默）
（递过去一个包袱）"拿着。"

解释: 用行动代替解释，让读者自己理解关系。
```

### 示例3：情绪直白

```
❌ AI版本:
"我真的很生气！你怎么能这样对我！"

✅ 人味版本:
"……"（攥紧拳头，指节发白，一言不发地转身就走）

解释: 让动作说话，读者自己感受到愤怒。
```

### 示例4：套路回应

```
❌ AI版本:
"受教了！师兄说得对！"

✅ 人味版本:
"……哦。"（心不在焉）

解释: 真实反应，不是每句话都有积极回应。
```

1. 写完章节后检查对话AI味
2. 角色对话听起来"都一样"时
3. 发现对话太顺、太有逻辑时
4. 想给角色加入独特声音时
