# 通用场景结构技法 (Universal Scene Craft)

> 从6部参考作品（术师手册/天启预报/谁让他修仙的/牧者密续/不许没收我的人籍/我师兄实在太稳健了）中提炼的跨作品通用场景结构模式。
> 这些是**结构级技法**，与具体风格无关，适用于所有网文创作。

---

## 一、设定说明场景（乒乓球模式）

> 出现频率：6/6 作品 ★★★

### 核心结构

```
信息点A → 角色反应 → 信息点B → 角色吐槽/追问 → 信息点C → 角色震惊/类比
```

### 通用规则

| 规则 | 说明 |
|------|------|
| **3段上限** | 绝不连续灌输超过3段设定无角色反应 |
| **反应多样化** | 吐槽、追问、类比、震惊、跑题——不能每次都是"哦" |
| **绑定角色个性** | 信息提供者应有鲜明人设（温柔导师/毒舌同伴/推销式热情） |
| **讲解即剧情** | 设定说明的过程本身也在推进剧情或揭示角色关系 |

### 变体

| 变体 | 适用场景 | 典型作品 |
|------|----------|----------|
| **导师问答式** | 师生关系、知识传递 | 牧者密续（马瑟斯讲课）、师兄（讲道） |
| **穿越者映射式** | 穿越/重生题材 | 牧者密续（游戏对号）、术师手册（前世类比） |
| **日常嵌入式** | 吃饭/工作中自然引出 | 谁让他修仙的（百香楼点菜）、天启预报（料理细节） |
| **演讲训话式** | 长辈训诫、组织动员 | 天启预报（林家训话）、师兄（凌霄殿） |

### 禁忌

- ❌ 连续灌输5段以上设定无角色反应
- ❌ 角色反应全是"嗯""哦""原来如此"
- ❌ 信息提供者像百科全书一样平铺直叙
- ❌ 讲完设定后对剧情毫无推动

## 二、战斗/冲突场景

> 出现频率：6/6 作品 ★★★

### 核心结构

```
对峙铺垫 → 试探交锋 → 劣势/危机 → 转机 → 爆发逆转 → 战后清算
```

### 通用规则

| 规则 | 说明 |
|------|------|
| **旁观者视角链** | 从低→中→高级旁观者递进评价，制造"全场震惊"效果 |
| **战中穿插** | 战斗不能纯打，必须穿插内心分析、旁人评论、吐槽 |
| **伤害有代价** | 每次战斗都应有明确代价（体力/装备/关系），不能无损通关 |
| **战利品/清算** | 战后必须有清算环节——收获什么、失去什么、关系变化 |

### 旁观者视角链模板

```
新手视角（"好厉害！"）→ 内行视角（"这个技巧……不可能"）→ 利益相关者视角（内心博弈+恐惧）→ 群体反应
```

每层旁观者的评价维度不同：
- 新手看力量
- 内行看技巧
- 利益相关者看威胁/机会

### 变体

| 变体 | 特征 | 典型作品 |
|------|------|----------|
| **暴力美学式** | 战前环境渲染、通感美学、暴力后骤然静寂 | 天启预报 |
| **游戏化式** | 战利品系统、技能名称、成长即时反馈 | 术师手册、牧者密续 |
| **策略式** | 以智取胜、分析弱点、团队配合 | 不许没收、师兄 |
| **考验通关式** | 标准答案+非常规解法+权威震惊 | 谁让他修仙的 |

## 三、日常温馨场景

> 出现频率：6/6 作品 ★★★

### 核心结构

```
场景设定(熟悉空间) → 角色聚集 → 轻松对话/互动 → 温馨细节定格 → 平静收尾(或被事件打断)
```

### 通用规则

| 规则 | 说明 |
|------|------|
| **情感锚点空间** | 固定一个"家"——小琼峰、猫老板餐厅、莫里亚蒂宅邸——所有外出冒险后必回 |
| **食物描写必备** | 食物是日常场景最有效的感官锚——做饭、吃饭、点菜、烤鱼 |
| **固定梗** | 角色间的重复性笑料（鸽子事件、消防安全等）制造亲切感 |
| **被打断的日常** | 温馨场景被突发事件打断→反差制造紧迫感，同时让读者更珍惜日常 |

### 日常场景的隐藏功能

| 功能 | 手法 |
|------|------|
| **节奏减速** | 大事件后的呼吸空间 |
| **嵌入伏笔** | 日常对话中不经意提及的细节，后续成为关键 |
| **关系推进** | 在放松状态下展现角色真实性格，推进情感线 |
| **世界观展示** | 通过食物/物件/习惯展示世界的生活层面 |

## 四、煽情/情感高潮场景

> 出现频率：5/6 作品 ★★

### 核心结构

```
情感铺垫（细节堆叠）→ 情感爆发点 → 冷静转折/吐槽打断 → 余韵留白
```

### 通用规则

| 规则 | 说明 |
|------|------|
| **3段上限** | 煽情绝不超过3段就必须被转折/吐槽/日常化打断 |
| **细节传递情感** | 不直说"他很感动"，用具体动作和物件传递——递茶、戳额头、无声叹息 |
| **日常化收尾** | 情感高潮后以朴素日常结束（"走吧去吃早饭"），比直接抒情更有力 |
| **沉默>语言** | 最有力的情感表达往往是沉默——沉默段落越长，情感重量越大 |

### 煽情后的打断方式

| 打断方式 | 效果 | 典型作品 |
|----------|------|----------|
| **吐槽打断** | 不消解情感，反而让情感更真实 | 术师手册、谁让他修仙的 |
| **日常化** | "走吧去吃饭"——将情感沉淀到日常 | 术师手册、天启预报 |
| **朴素收尾** | 环境描写/行为描写代替抒情 | 天启预报（"该干什么干什么去吧"） |
| **时间跳跃** | 跳过直接反应，展示"后来"的变化 | 师兄 |

## 五、说服/谈判场景

> 出现频率：5/6 作品 ★★

### 核心结构

```
分析目标心理 → 建立信任/共情 → 渐进引导（层层递进）→ 击中核心 → 对方动摇 → 达成目的
```

### 通用规则

| 规则 | 说明 |
|------|------|
| **先共情后逻辑** | 先让对方感到被理解，再用利害分析推进 |
| **适度暴露私心** | 承认自己"也并非毫无私心"→增加可信度 |
| **给假选择** | 看似给出多个选项，实际上只有一个最优选——逻辑压迫 |
| **不用威胁** | 语气始终温和平等，威胁感来自逻辑本身而非言辞 |

### 变体

| 变体 | 特征 | 典型作品 |
|------|------|----------|
| **以退为进型** | 先示弱→再利害分析→让对方主动帮忙 | 师兄（李长寿） |
| **话疗式** | 先共情→给选择→逻辑封闭 | 不许没收（兰奇） |
| **冰山式** | 表面寡淡→水面下信息量巨大 | 天启预报（艾晴与阴崖） |
| **社畜话术式** | 用职场逻辑/谈判技巧 | 术师手册（亚修） |

## 六、身份揭示/反转场景

> 出现频率：5/6 作品 ★★

### 核心结构

```
预期建立（角色自信/读者假设）→ 信息积累 → 反转揭晓 → 读者/角色震惊 → 重新评价
```

### 变体

| 变体 | 手法 | 典型作品 |
|------|------|----------|
| **先知后行型** | 读者先知真相，看角色在错误信息下行动→戏剧反讽 | 谁让他修仙的 |
| **棋局揭示型** | 数十/百章前伏笔→现在揭示当初行为的深意 | 师兄 |
| **权威解构型** | 史诗→草率，传说→日常——高大上预期被生活细节打破 | 谁让他修仙的 |
| **身份伪装型** | 伪装日常→可疑试探→危机→化解→继续伪装 | 牧者密续、不许没收 |
| **冰山渐露型** | 对话中逐步暗示→读者比角色先猜到→最终确认 | 天启预报 |

## 七、考验/试炼场景

> 出现频率：4/6 作品 ★

### 核心结构

```
考验规则说明 → 标准答案示范 → 非常规解法 → 权威震惊/强行解释 → 收获/评价
```

### 通用规则

| 规则 | 说明 |
|------|------|
| **多重路径** | 同一考验展示不同角色的不同解法——展现角色差异 |
| **标准vs非标准** | 至少一个角色走标准路线（参照物），主角走非常规路线 |
| **权威震惊反应** | 考官/长辈的震惊反应是重要的爽点来源 |
| **信息密度高** | 规则说明本身就是世界观输出，需用乒乓球模式消化 |

## 八、政治博弈/审判场景

> 出现频率：4/6 作品 ★

### 核心结构

```
多方入场（身份/立场）→ 表面寒暄（暗藏试探）→ 议题展开 → 核心冲突暴露 → 话术交锋 → 裁决/新格局
```

### 通用规则

| 规则 | 说明 |
|------|------|
| **至少三方** | 博弈必须有3方以上利益交织才有张力 |
| **表里不一** | 每句对话标注"表面含义"和"真实意图"——通过视角角色的分析呈现 |
| **物证/反转** | 关键时刻亮出预先准备的证据/底牌 |
| **旁观者解说** | 旁观者的内心分析帮助读者理解博弈态势 |

## 场景组合节奏建议

单章不应只有一种场景类型。推荐的组合模式：

| 组合 | 效果 |
|------|------|
| **日常 → 事件触发 → 行动 → 回归日常** | 安全的基本循环（最常用） |
| **设定说明 → 战斗验证** | 刚学的知识立刻派上用场→爽感 |
| **煽情 → 日常化 → 伏笔暗示** | 情感沉淀后留下新悬念 |
| **说服 → 棋局揭示** | 成功说服后揭示背后更大的布局 |
| **试炼 → 旁观者评价 → 日常** | 实力展示+社交影响+减压 |
