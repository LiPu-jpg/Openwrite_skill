# 通用对话技法 (Universal Dialogue Craft)

> 通用技法层：跨作品通用的对话写作原则。

## 对话格式规范

### 中文小说标准

| 元素 | 格式 | 说明 |
|------|------|------|
| 普通对话 | "" 双引号 | 出声说话 |
| 心灵对话 | 「」或『』 | 心声、telepathy |
| 引用/强调 | 「」 | 引用他人话语、术语强调 |

### 对话标签

**常用标签**：说道、道、问道、喝道、喃喃、低声道、冷声道

**标签位置**：
- 前置：「亚修说道："我不知道。"」
- 后置：「"我不知道。"亚修说道。」
- 省略：连续对话中靠上下文区分

## 对话节奏光谱

| 类型 | 特征 | 适用场景 |
|------|------|---------|
| 短促对话 | 1-5字/句，快速交替 | 审讯、争吵、战斗交锋 |
| 标准对话 | 10-30字/句 | 日常交流、关系展现 |
| 长篇对话 | 50字+/句 | 演讲、仪式、重要说明 |
| 独白 | 连续多句无打断 | 极少使用，仪式性场景 |

## 对话标签省略规则

### 可省略场景

1. **两人快速对话**：交替明确，无需标签
2. **语气特征明显**：角色有专属口癖/语气词
3. **动作穿插**：动作描写已暗示说话人

### 必须标签场景

1. **三人以上对话**：需要明确说话人
2. **长时间沉默后**：重新建立说话人
3. **情绪转变点**：标签承载情绪信息

## 信息传递技法

### 通过对话引入设定

```
角色A提问 → 角色B解释 → 自然引入世界观
```

**优点**：避免作者直接说明，保持沉浸感

### 对话展现关系

| 关系类型 | 对话特征 |
|----------|----------|
| 上下级 | 指导式、命令式、敬语 |
| 平辈 | 试探式、玩笑、直呼其名 |
| 亲密 | 省略主语、默契、私密话题 |
| 敌对 | 讽刺、威胁、冷淡 |

### 对话与内心交替

```
对话（外部事件）→ 内心独白（真实想法）→ 对话（继续推进）
```

**效果**：展现角色表里不一，增加层次感

## 乒乓球规则（核心）

信息提供者每说1-2个信息点，接收者必须有反应：

| 反应类型 | 示例 |
|----------|------|
| 追问 | "等等，你说什么？" |
| 吐槽 | 内心：这也太离谱了吧 |
| 表情变化 | 他的眉头皱了起来 |
| 动作反应 | 她下意识后退一步 |
| 重复确认 | "你是说……真的？" |

**禁忌**：连续灌输超过3段无反应（读者会失去代入感）

## 群体对话技法

### 众声喧哗

多人场景中，不标注具体说话人，制造群体氛围：

```
"快看！"
"怎么可能？"
"我早就说过了……"
人群中议论纷纷。
```

**标签选择**：有人笑道、旁边有人说、人群中传来声音

### 焦点切换

群体场景中，用具体标签将焦点拉回主要角色：

```
众人议论 → "够了。"李明冷声道。（焦点回归）
```

## 语气词系统

### 通用语气词

| 语气词 | 情绪 | 使用场景 |
|--------|------|---------|
| 嗯 | 肯定/思考 | 回应、沉思 |
| 啊 | 惊讶/恍然 | 发现、理解 |
| 哦 | 了解/冷淡 | 回应、敷衍 |
| 嘶 | 思考/疼痛 | 沉思、受伤 |
| 哼 | 不满/轻蔑 | 生气、鄙视 |
| 呵 | 冷笑/无奈 | 讽刺、自嘲 |

### 角色专属语气词

为重要角色分配专属语气词/口癖，作为声音标识。

## 对话诊断清单

- [ ] 连续对话是否超过5轮无动作/心理穿插？
- [ ] 信息灌输是否超过3段无反应？
- [ ] 三人以上场景是否有足够标签？
- [ ] 角色声音是否有区分度？
- [ ] 对话是否在推进剧情或展现关系？（无意义闲聊？）
